Hi, i had resolved this problem yesterday. In fact, the FHoSS was upgraded the last period so try to install a jdk 1.6 instead of jdk1.5. You should verify that when rebooting the system that the following command return the path of the jdk installation: 

#echo $JAVA_HOME

/path/jdk1.6xxx

If doesn't work, i advice to  to recuparate the old version of FHoSS and reinstall the jdk 1.5 and modify the JAVA_HOME(obligatory) and restart the system.

Good luck, if following this steps, it should be work.
