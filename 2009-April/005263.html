The IMS core should work in both IPv4 and IPv6. Making it work in 
parallel would be a bit tricky though, but not impossible (the default 
config files were thought with just one environment in mind).

1. yes, it would work, provided that somewhere in the IMS Core a 
component would have both IPv6 and IPv4.

2. no. if the core uses just IPv4, it can not be contacted by IPv6-only 
clients properly. So at least enable a P-CSCF which has a dual IPv4/IPv6 
setup.
