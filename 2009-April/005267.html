Hi Laura,

The missing Authorization header is not a problem for MD5 clients. But 
then please configure the S-CSCF and HSS for MD5 authentication.

Then I don't think that Asterisk should act as a client in regard to 
IMS. In my opinion it is similar to an Application Server, so it should 
not register, but be triggered when required.

Cheers,
-Dragos
