Hi all,

In case you were wondering about the last commits, we are working on 
this as part of a project related to Emergency services in IMS: 
http://www.ict-peace.eu/

The P-CSCF, S-CSCF and HSS in the trunk have been upgraded to support 
these services and now further work on realizing the E-CSCF and the 
components around is under-way in the |12_em_services branch.

If you want to know more and give it a try, please see this page: 
http://www.openimscore.org/emergency . There you have more explanations 
about the architecture as well as installation and simple testing 
guides. As usual we would appreciate any feedback and we would welcome 
any contributions in this area.

Cheers,
-Dragos
