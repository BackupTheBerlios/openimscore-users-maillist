Hi,

For the web interface issue have a look at this thread:

https://lists.berlios.de/pipermail/uctimsclient-users/2009-February/000493.html

To see the diameter messages in Wireshark you need to include the relevant
interface dictionaries (Rx and Gx).  See the README_PCC file in the docs
folder of the cdp_05_sessions CSCF branch.

I have noticed that even once doing this I need to decode the messages in
Wireshark as Diameter messages to see all the fields.
