Hello,


I attach the scripts I use with OpenIMSCore (version 1)

your server must have all pre-requisites (dns, ipsec ...) as script does 
not deal with this matter yet.

configure dns script in tools/oic_dns_tools directory to adapt to your 
network - default 192.168.0.*

copy your FHoSS and ser_ims sources in trunk directory

launch with ./cli (otherwise you can use each script independently if you 
want [in tools directory])

installation and domain config is adapted to default OpenIMSCore settings 
(/opt/OpenIMSCore ....)

modify paths in tools/oic_domain_adapter and tools/oic_installer scripts
