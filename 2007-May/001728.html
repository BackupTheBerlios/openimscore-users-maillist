Hi Daniel,

to the 2. question:

Charging issues is specified in specification serial: 32.240 (overview),
32.260 (ims charging), 32.299 (diameter charging application). 

Currently, the offline charging interface at CSCFs is being implemented
(working branch: /branches/05_cdp_sessions). In order to observe message
exchange, you need install a CDF (Charging Data Function) that interacts
with CSCFs. Unfortunately, the CDF is not a component of the openimscore
project.
