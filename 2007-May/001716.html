Hi Fei,

I can tell you that you need much more than 1000 subscribers :-). Then
it depends on what you understand by handle. To just register more, you
need to increase the memory that the CSCFs are allowed to use.

Please keep one thing in mind: to see the real performance you will need
to use different IP:port pairs for each user as it would be in the real
case, both in the Contact headers and in for sending/receiving messages.
I don't think that the current SIPp is able to control this.
