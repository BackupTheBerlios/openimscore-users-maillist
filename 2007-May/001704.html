Hi
 
While starting the FOKUS CSCF application, following warning is
observed:
"WARNING: fix_socket_list: could not rev. resolve 10.0.64.50"
 
This warning comes because reverse DNS is not set.
 
Reverse DNS can be setup as follows:
1.	Configure a file 10-0-64.dnszone with following contents
$TTL 1W
@                       1D IN SOA       localhost. root.localhost. (
                                        2006101001      ; serial
                                        3H              ; refresh
                                        15M             ; retry
                                        1W              ; expiry
                                        1D )            ; minimum
 
                        1D IN NS        ns
ns                      1D IN A         10.0.64.18
 
50                      PTR             pcscf.open-ims.test
50                      PTR             icscf.open-ims.test
50                      PTR             scscf.open-ims.test
18                      PTR             hss.open-ims.test
 
2.	Add an entry into the named.conf file
zone "64.0.10.in-addr.arpa." IN {
        type master;
        file "10-0-64.dnszone";
        allow-update { none; };
};
 
3.	Restart the named service
 
If something is missing, please provide you inputs/comments.
 
Regards,
Gaurav Kansal
