Hello b p,

As the error says, you need Java 1.5 to build/run the HSS. This is
written in the prerequisites (maybe you missed it).

Even if you have installed Java 1.5, please make sure that the one that
you are actually using is the right one by running "java -version". It
could be that you are still using an old installation.

-Dragos
