There is no valid option as "Digest-MD5" in the scscf.cfg. Only MD5
(CableLabs-Digest - but this is not yet implemented).

I don't know what it is wrong with you registration and it is impossible
for me to debug your client. The Open IMS Core has been reported as
working correctly with MD5 client and I doubt that there is a problem
there. I am attaching a correct MD5 registration. Make sure that you use
the same field for the Request-URI, uri, realm, username, etc.

-Dragos
