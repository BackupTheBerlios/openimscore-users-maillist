So, yes, there is another HSS developed at FOKUS besides FHoSS and it is 
called CHeSS. It just recently received a shiny new platform and a lot 
of upgrades (Sh, Sp interfaces, new GUI) in the context of the OpenEPC 
project ( www.openepc.net ). Also some serialization bugs have been 
discovered lately and fixed, which should further improve the 
performance. Last time we did some performance checks, we could do 
around 250 registrations per second with it, which was pretty much what 
a simple MySQL backend could handle (which is actually quite a lot 
translated in number of queries/sec).

If anyone is interested in this, let me know directly.

Cheers,
-Dragos
