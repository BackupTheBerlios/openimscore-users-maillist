Hi,

there is no component for SS7 Interconnection in the OpenIMSCore.
You'll need some sort of SIP-to-SS7 Gateway, if you want to spend
money, you can buy them from Cisco or you can choose a cheaper
solution like the Yate SS7 Gateway (it's Open-Source, see
http://yate.null.ro/pmwiki/index.php).
Asterisk has a SS7 Implementation too, but i am not sure about the
quality (see http://www.voip-info.org/wiki/view/Asterisk+SS7).
