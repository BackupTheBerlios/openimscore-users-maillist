Don't worry about what the Open IMS Core uses internally. It decides
automatically based on the message size. So if it is small enough, it
uses UDP as UDP is faster.

The Record-Route and Via headers are not internal to the IMS network.
There would be too much overhead to store the Via headers in each node
for each transaction. Record-Routes (and also Via headers) should be
saved in the P-CSCF (only) for enforcing correct traffic. However, they
should never be removed.

-Dragos
