Hello All,

you'll find a new script(configurator.sh) at ser_ims.
 
This lets you change default domain name "open-ims.test" and ip address
"127.0.0.1" in config files to <your_domain_name> and
<your_ip_address>. 

It is thought to make the work easy for users who want to run the
software with alternative domain name and ip(saves time). 

Usage-
Copy all *.cfg , *.xml and *.sql files into the same directory of
configurator.sh, execute it. Then follow the instructions. 

I'll be writing a detailed -howto- about the usage. You'll find it on
the website soon.
