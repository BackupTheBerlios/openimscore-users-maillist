The Early IMS Authentication bases itself on network
components asserting that IP addressing are and can
only be used by authorized identities. In this case
there is no need to challenge the REGISTER requests.
However, it is often hard to distinguish between clients.

The CSCFs and HSS need to be modified accordingly to
support these features. References: TS 33.978.
