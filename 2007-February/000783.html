Hello Luis,

Using the java version is easier because of Java ;-). However, in terms
of performance, although the Java version performed in our tests better
than other C/C++ stacks, it is worse with performance than the C version.

The Java version was written by me and Shengyao Chen and the C version
also by me. So you will find them very similar.

Also, keep in mind that they are not fully featured 100% compliant. So
choose one version with which you would be comfortable in doing changes.

-Dragos
