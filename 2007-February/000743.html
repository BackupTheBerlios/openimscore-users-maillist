In your configuration file, the respective peers are commented.

So you need elements like this:

<Peer FQDN="peername.peerdomain.com" Realm=" peerrealm.com" port="3868"/>
//
Then periodically the JavaDiameterPeer will attempt to connect to this
peer, if not connected already.

-Dragos
