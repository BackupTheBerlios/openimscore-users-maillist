Hi all

This month's UCT IMS Client release (1.0.5) supports a few new features 
worth mentioning, most notably a very basic implementation of XCAP support.

After the positive feedback from last month's presence How-To we will 
stick to this format for now. On the website 
(http://uctimsclient.berlios.de) I've posted an XCAP server How-To to 
get you started with your XCAP applications. This month we're focusing 
on presence rules, but once you've got this going, moving on to RLS and 
network-stored address books will be easy. Users of Eyebeam (not X-lite 
unfortunately) will be glad to hear that it also supports XCAP.

The Open Mobile Alliance typically refers to these servers as XML 
Document Management Servers (XDMS). See their website for more details:
http://www.openmobilealliance.org/release_program/Presence_simple_v1_0_1.html

There's also a basic Back-to-back user agent How-To so you can setup a 
click-to-dial website.

As always send any feedback, corrections and additions to the mailing list.

Regards

Dave
