Take a look at the scscf.cfg file also and change the default
authentication method.

Oh, and you might want to upgrade your whole installation as the latest
ser_ims + the FHoSS/branches/01_RevampDB branch support a LOT more
algorithms, and extra you can let the HSS decide which auth scheme to
use for each user.
