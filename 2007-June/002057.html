Hello fengbin,

reverse DNS is needed, between others, when you use for example an IP
Address in a Route header and CSCF needs to identify if that IP address
is itself. The simple 127.0.0.1 default setup does not require the
reverse DNS, but if you would move to other IP addresses it will be
required.

-Dragos
