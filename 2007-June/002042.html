I should say first that there is no quick&dirty hack for this.

Then I was trying to find out the status from other people that told me
that they were working on the Ro, but no reply so far.

In a nutshell:
- you need to load the CDiameterPeer (cdp) module from pcscf. There are
already 2 examples in the icscf and scscf modules for the Cx interface
- you need to write your Ro logic - there are examples for Cx, of course
- you need to build a PDF server, if you don't have one - you could use
either the standalone CDiameterPeer server or JavaDiameterPeer

Hope this helps... I am not sure what you mean by
static/dynamic/temporary...
