Hi Li,
Ubiquity and weblogic SIP servlet API's have one method called doRequest(). 
We can use doRequest() method like below.
public void doRequest(SipServletRequest req) throws ServletException,
			IOException {
		if(req.getMethod().equals("PUBLISH")){
			doPublish(req);
		}
		else{
			super.doRequest(req);
		}
	}
This little information will help you much.
Ubiquity and weblogic SIP servlet API's have doRequest() methods.
