By default it stores everything in memory, but you can enable (in the
configuration) persistency to files or database. For now it just dumps
all the memory contents to files or database every configurable
interval. No support yet for a "caching mode", where the CSCFs would
hold in memory upto some amount of state (dumping to DB after a
configurable expiration time) and query the DB when needed.
