Hi Raman,

the Early-IMS authentication was added in a (for now simplified) version
lately. 

You can enable it by setting "registration_disable_early_ims" to "0" in
scscf.cfg and please ensure as well that the auth scheme is selected for
the user that you are about to register in the HSS web console plus that
you provide the IP address of the UE in the web interface. 
I am unsure what client you are using, but be warned that this one
should not send an authorisation header.

Optionally, you can also just use the MD5 authentication scheme (also
avoids AKA procedures). Yet, there will always be some form of
authentication ;)

Regards,

Peter
