Hi Kiran,

> I would like to connect 2 OpenIMSCore each in different domains(ex xyz.com
> and abc.com), to test the roaming scenario's and also the call setup
> scenarios between 2 IMS domains.

> Please give me directions as to how this setup could be done.

I have already done this. It is a straightforward task. You have to
implement 2 OpenIMSCore platforms running on the same or different computers
(I for myself have done it on my notebook) but you have to use different DNS
domains for both networks as you mentioned above.

Then you setup calls and it should work.
As a further step you can enable roaming so that subscribers can attach in
the foreign network. This has to be done in HSS. 

This should work easily.
