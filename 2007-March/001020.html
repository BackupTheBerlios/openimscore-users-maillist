Date: 2007-Mar-03 02:17
By: vingarzan

Comment:
The client should use the P-CSCF as strict outbound proxy. As such, it does not need to insert a Route header indicating the P-CSCF in initial requests. As per 24.229, the initial requests should contain a Route header with the exact values received in the Service-Route header at registration.
