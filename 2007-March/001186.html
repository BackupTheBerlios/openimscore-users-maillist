> Is there any documentation for adding service profiles, trigger points and
> ifc's etc. and other necessary configuration changes in open-ims!

No, at least I don't know. But with some basic understanding how iFCs are
organized it should not be too difficult. There is a certain logical chain:

You have to define trigger points, then you need to bind those trigger
points to an initial Filter Criteria (iFC) and then add the iFCs to a
service profile. The service profiles then get assigned to IMS Public IDs.

Then it will work.
