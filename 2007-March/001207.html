The whole procedure looks like:

In the menu Service Profiles:
1. Create an Application Server
2. Create a Trigger Point
3. Create an iFC using info from steps 1 and 2 above
4. Create a Service Profile and assign the iFC from step 3

In the menu User Profiles - Public Identity:
5. Assign the Service Profile from 4 to the User Identity
