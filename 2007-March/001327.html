> > > 1.Im curious as to which entity CFB AS would query in order to select
> > > the next contact to call, if previous one was busy... Are there any
> > > specs available? Diagrams? Call flows?

The actual spcification for Call Forwarding for fixed network (ETSI TISPAN
R1) is ETSI TS 183 004: PSTN/ISDN simulation services: Communication
Diversion (CDIV). The spec can be downloaded from ETSI via
http://portal.etsi.org/Portal_Common/home (use menu button Approval ->
download)

> > > 2. If no specs available, then could you tell me, which AS would be
> > > the easiest to implement first with OpenIMSCore? And again... Any
> > > specs/diagrams/flows?

Maybe an easier spec for beginning might be ETSI TS 183 007: PSTN/ISDN
simulation services; Originating Identification Presentation (OIP) and
Originating Identification Restriction (OIR).

An overview of all PSTN/ISDN simulation services is: ETSI TS 181 002
Multimedia Telephony with PSTN/ISDN simulation services.

These are the PSTN/ISDN simulation services, but there are of course also
other services as well.
