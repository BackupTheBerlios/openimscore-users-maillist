The error comes from the HSS because you probably forgot to add the
private & public identities there.

The SIP2IMS will be deleted maybe today because it seems to be causing
WAY more trouble than benefits. The S-CSCF supports MD5 even without it.
All you need to do is either:
- change the scscf.cfg file to perform MD5 by default
- add an Authorization header with algorithm=MD5 in the first REGISTER
request to indicate your preference for MD5.

-Dragos
