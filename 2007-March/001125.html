Hi Andrey,

In order to change the domain name and ip address you can use a script(configurator.sh) 
at ser_ims/cfg/. Copy the all *.sql *.cfg *.xml files that you want to modify to the same
directory as configurator.sh and execute it. Follow the instructions then you should have
your files reconfigured for new domain and ip.
