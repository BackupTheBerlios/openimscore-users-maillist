Hi Alessandro,

How did you compile ser_ims? Did you follow the instructions? 
If you do a "make install-libs all" this shouldn't happen...

The reason that FHoSS gives you error that you haven't started
the scscf at that time. After you start FHoSS, start two terminals,
Type in both:
cd /opt/OpenIMSCore

Then type in first terminal:
./icscf.sh

in second terminal:
./scscf.sh

If your dns setup is correct, you wont read those ERROR messages
at FHoSS anymore.
