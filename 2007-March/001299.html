You probably did not start your DNS server or configured in wrongly it
in resolv.conf or maybe you have a firewall? In your trace you can see
that in packets 9-40 the P-CSCF is trying a DNS query and that fails as
the configure DNS server to be actively rejecting the queries.

As Peter said, please check your DNS installation. You could use the
"dig" utility to check for functionality.
