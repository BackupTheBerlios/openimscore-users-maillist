Sam,

Your Java is correctly installed, but when you are executing javac, you 
are not executing it from the right installation. Linux comes with a GNU 
Java bundled with it, which will not do the job you need.

Edit your /etc/profile file on unix and add the path to your NEW java 
installation to the PATH. Something like this:

PATH=/usr/local/java/jdk1.6.0/bin:$PATH

Then logout and login again. If you do which javac it should then point to 
the path to your JDK installation.

regards,
Josef
