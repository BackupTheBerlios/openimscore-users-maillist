Hi All,

I have been trying to get IMS Communicator ( downloaded binaries for linux )
to OpenIMS using AKAv1_MD5, but havent got it to work yet.

The steps I followed were :

1 > Set the authentication type to AKAv1-MD5 in the scscf.cfg. Do I need to
change anything in the hss.properties file ? Start the core.
2 > Change the HSS entries in the Web Console for this user to AKAv1-MD5.
3 > Set the IMS Communicator values to :

<ims>
                <IMS_CLIENT value="true"/>
                <PRIVATE_ADDRESS value="bob at ims.gtisc.gatech.edu"/>
                <PREFERRED_ADDRESS value="bob at ims.gtisc.gatech.edu"/>
                <PREFERRED_DISPLAY_NAME value="bob"/>
                <ACCEPT_REFER_WITHOUT_SESSION_ON value="true"/>
                <REFER_TIMEOUT value="60000"/>
                <REG_EVENT_SUBSCRIPTION_TIMEOUT value="600000"/>
                <ACCESS_TYPE value="IEEE-802.11"/>
                <DEFAULT_LOCAL_PRECONDITION value="sendrecv"/>
                <VIDEO_BANDWIDTH value="75"/>
                <AUDIO_BANDWIDTH value="25"/>

                <!-- testing feature -->
                <PCSCF_DYNAMIC_DISCOVERY value="false"/>

                <sec>
                    <PRIVACY value="none"/>
                    <AUTH_ALGORITHM value="AKAv1_MD5"/>
                    <!-- operator ID : 16 bytes (32 hex char) -->
                    <OPERATOR_ID value="00000000000000000000000000000000"/>
                    <!-- if SECURITY_AGREEMENT value is empty,
Security-Client header is not sent and IPSec not setup -->
                    <!--SECURITY_AGREEMENT value="ipsec-3gpp"/-->
                    <SECURITY_AGREEMENT value=""/>
                    <SECURITY_ALGORITHM value="hmac-sha-1-96"/>
                    <SECURITY_ENCRYPT_ALG value=""/>
                    <SECURITY_SPI_C value="10002"/>
                    <SECURITY_SPI_S value="10004"/>
                    <!-- if SECURITY_PORT_C empty, port-c parameter is
filled with PREFERRED_LOCAL_PORT-->
                    <SECURITY_PORT_C value=""/>
                    <SECURITY_PORT_S value="5064"/>

                    <REJECT_MALFORMED_NONCE value="false"/>
                </sec>
            </ims>

 However the authentication fails. Are my steps correct or have I missed out
any step ? Please let me know.
