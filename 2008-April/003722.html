I've just been following the install guide at:

        http://www.openimscore.org/installation_guide

and had a couple of suggestions for the guide.

It seems that the build requires flex and bison, though these are
not mentioned as software prerequisites. I guess they would fit in
the list nicely beside gcc and jdk in Step 1? It probably also
requires GNU make, but maybe that's too obvious ;-)

In setp 3, in the section FHoSS there is a "cd.." that should be
"cd ..".
 
In step 6, when starting the HSS startup.sh script, the install
guide does not mention that you need the JAVA_HOME environment
variable set or that you should be in the  FHoSS/deploy/ directory.
