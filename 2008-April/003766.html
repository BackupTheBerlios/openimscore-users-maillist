Hi,
 
I just want to report here a small issue that is already mentionned in
the archive but not clearly answered.
When entering a new IMPI/IMPU in the database and that there is no way
to successfully register with the 403 error message + "HSS roaming not
allowed on new account" it might be because the list of visited network
has not been set.
 
In the IMPU table, select the right domain in the "Add Visited-Network":
it should be open-ims.test with a fresh install of the server, add it
and then the domain will appear in the List of Visited Network. Don't
forget to save.
