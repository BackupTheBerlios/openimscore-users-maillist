Hi Tomasz,

considering that OpenIC Lite does not implement the attachment 
procedure, then yes, it is on purpose. If you have any suggestions on 
how to do it in a platform-trouble-free manner and also keep it jdk 1.3 
compliant for Windows Mobile, we'll add it...

So OpenIC Lite for now has a simple strict outbound proxy configuration.

Anyway, do you really control the Access Network to be able to actually 
use such a feature? Then OpenIC_Lite is just an appetizer and is more 
suitable for use without encryption, for debugging reasons... if you 
need all those features, you should contact the OpenIC developers for a 
full version with all the bells and whistles....

Cheers,
-Dragos
