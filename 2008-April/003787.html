Hi Tomasz,

have you checked the IFC XML output of the HSS on user's registration if 
anything is missing in the IFC?

While ago I had a similar problem and figured out, that the INVITE is 
not set correctly on IFC creation. In this case try to create your IFC 
using REGISTER method, save it and change it to INVITE again. It solved 
the problem at my side and I guess since the INVITE is the default item, 
is saves it wrong.
