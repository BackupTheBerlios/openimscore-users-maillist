Hi Persa,

This situation  is not supported because it is impossible then for the 
P-CSCF to tell which user is which. So if both A and B share the same 
port, how can the P-CSCF prevent A from impersonating B?

If you would just want to use multiple identities on the same contact, 
then I guess you'd better use the implicitly registered identity sets.

Cheers,
-Dragos
