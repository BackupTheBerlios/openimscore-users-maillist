Hi David,

Open IC Lite works, without modifications - all you need is to change 
the P-CSCF address and it's DNS entry to a IPv6 address.

So should any other Java client, as I don't think that there are 
differences in the code for IPv4 or IPv6. For C clients then the 
differences are not big, just that the new and proper name resolution 
functions should be used. From my experience it's all about how you 
resolve strings containing FQDNs or addresses into data structures 
containing the addresses. Other than that, the networking code is the 
same and at the SIP protocol level no changes should be required.

Cheers,
-Dragos
