I'd like to clear it. 

182 027 & 182 028 are Stage 2 documents. (Stage 2 is Architecture)
(anyone can download from http://pda.etsi.org/pda/queryform.asp )

182 027 seems to be IMS Model over NGN to use SIP(P-CSCF).
182 028 seems to be Non-IMS Model over NGN to use RACS by Gq' interface. 

Of course, either model is substantially accessed to RACS by Gq'
interface. 


03127-ngn-r2v202.pdf is Stage 3 document. (Stage 3 is Signalling)
It seems to be IMS Model over NGN.


I think there are some models ....
 IMS Model over NGN
            ---> using SIP over NGN model
 Non-IMS Model over NGN 
            ---> define new subsystem on NGN for IPTV, not SIP
 Non-NGN model over managed network
            ---> IPTV model has already been offered by some carriers
 Non-NGN model over non-managed network
            ---> Internet model (without QoS)

It is confused for a moment. 
I am interested though I do not know which becomes a main current.
