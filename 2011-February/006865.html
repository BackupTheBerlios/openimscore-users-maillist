Dear Alex Atmana,

For using OpenIMSCore on IPv6 there is not special building required. The tutorial from http://www.openimscore.org/installation_guide is enough.

The only thing that you need is to configure OpenIMSCore to use IPv6 addresses for binding and forwarding.
The needed hints can be found here: http://www.openimscore.org/node/125

Btw, make sure that the kernel you are using is handling well IPv6 addresses and DNS on IPv6.

Best regards,
Ancuta Onofrei
