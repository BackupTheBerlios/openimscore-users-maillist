Trying to query multiple user data in one UDR from the
FHoSS by adding multiple "Data-Reference" AVPs to the
UDR (e.g. 'S-CSCFName'  and 'IMSUserState') results in
getting only the first requested one in the result.

This feature was only added in Rel.7 of 3GPP TS 29.329
and should be supported in the coming months.
