 Hello all,

 The FHoSS has to be started from the deployment folder (implicitly is "deploy"), not from "scripts"! "deploy" folder is created after "ant deploy". There, the classpath is setted accordingly and it works fine. Also, it should be mentioned that some libraries needed by tomcat are contained only in deploy/lib and are not in "lib" of the base directory of FHoSS. These are taken from tomcat/lib and copyed in deployment. 
Please use the script from deploy, and it should work without problems.
