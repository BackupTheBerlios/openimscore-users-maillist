>  It seems that we have a bug here...it is a problem that relates to hibernate sessions. I have posted a bug on berlios here: https://developer.berlios.de/bugs/?func=detailbug&bug_id=9538&group_id=7589, so you will see the status of this bug, and also the solution, when will solve this. 
> I am currently working in solving this bug, so I hope it will be solved very soon.
>
>  Until then, a solution for running FHoSS if this problem persists also after a restart of FHoSS, is to reinstall the database:
>
> From mysql shell:
>  drop database hssdb;
>
> and then run again the sql scripts:
>  mysql -u admin -p <hssdb.sql
>  mysql -u admin -p <userdata.sql
>
> After this, restart FHoSS and try again. I know that is not the best solution, but this works if your database was corrupted by hibernate.
