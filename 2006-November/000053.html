    Hello Jose,

 If you want to do "ant gen" when you do not have an internet connection you should solve this problem like this:

1 - download the xml.xsd from here: http://www.w3.org/2001/xml.xsd and copy it on your machine, for example in xsd/ folder of FHoSS

2 - edit the file xsd/ZhDataType.xsd and change the path "http://www.w3.org/2001/xml.xsd" to your local path; for example: "file:///opt/openimscore/FHoSS/xsd/xml.xsd"

e.g. 
Original:
 <xs:import namespace="http://www.w3.org/XML/1998/namespace"
      schemaLocation="http://www.w3.org/2001/xml.xsd"/>

Modified:
 <xs:import namespace="http://www.w3.org/XML/1998/namespace"
      schemaLocation="file:///opt/openimscore/FHoSS/xsd/xml.xsd"/>

3 - run ant gen and it should work :)
