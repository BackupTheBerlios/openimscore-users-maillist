Hi,
 
> Now i achieve to send the UDR request and i can see it with Wireshark, but
> i have the following error : Diameter_Error_User_Data_Cannot_Be_Read .
> How can i correct that?

According to TS 29.329 the error means: The requested user data is not
allowed to be read.

So you have to enable access to this data via HSS (Sh Interface -
Permissions).
