> Can anybody tell me how to add a new visiting network through HSS console
> or other means ?

This has to be done via HSS GUI and requires two steps:

1. In the Network Configuration menu you can create a new visited network.

2. In the menu for Public User Identities you can add the visited network
thus enabling roaming.
