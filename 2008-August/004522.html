Hi Teck

Have a look at the files in the bind directory - e.g.
/etc/bind/open-ims.dnszone . This contains lines like

pcscf                   1D IN A         127.0.0.1

where the first part of the domain name is defined - the rest of the name is
defined elsewhere.

Either you have to get the other address's added to your company's DNS
server (pointing at the relevant ip - can be the same as in your email if
you are running all the elements on one machine) or you have to set up your
own domain name server to resolve these names. - You can try and base this
config on what is already there, but will be easier to do if you find a
generic DNS tutorial.

Hope that helps
