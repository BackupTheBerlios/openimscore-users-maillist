But then, please also update the DNS for the domain, so that your 
inter-domain calls won't end-up in the P-CSCF instead of the I-CSCF... 
Then still, many nodes would not make SRV queries and use the default 
5060 port when trying to contact an I-CSCF... so I really don't think 
that changing the I-CSCF port is a good idea. Instead I would suggest to 
move the P-CSCF to another IP and port 5060 if you have trouble with the 
client. Again, don't forget to update the DNS entries for the changed IP 
addresses.

Cheers,
-Dragos
