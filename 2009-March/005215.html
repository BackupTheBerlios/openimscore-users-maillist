There is obviously a problem with the killser script which is called before
starting the p-, i- and s-cscf. It kills x-cscf processes from a previous
session which might be running. 

After installation the script along with "stopser" is usually in
/opt/OpenIMSCore/ser_ims/cfg. So check where it is and perhaps adapt your
pcscf.sh script.
