Devdutt,
 
> Could you also please tell me more about the other options on the trigger
> points such as Term-Reg/Unreg, Orig-Unreg.

Term-Reg means: the Triggerpoint is valid for terminating sessions if the
user is registered
Term-Unreg means: the Triggerpoint is valid for terminating sessions if the
user is not registered
Orig-Unreg means: the Triggerpoint is valid for originating sessions if the
user is not registered

More details about Trigger Points and Service Point Triggers can be found in
TS 29.228 / B.2.3.
Further information about Filter criteria can also be found in TS 23.218 /
6.9.2
