The vmware image has the CSCFs and the HSS configured as an 
automatically started service - see /etc/init.d/openimscore. You can use 
this to start/restart/stop/etc the core. If you want to change the 
automatic start on start, you can use the rc-update utility for 
controlling which services are started when.

Cheers,
-Dragos
