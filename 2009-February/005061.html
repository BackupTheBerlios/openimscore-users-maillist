Your core is not in a position to offer its subscribers and users a wide
variety of services; it is limited in what it can offer. App. Servers then
come into the picture with the capability to offer almost any conceivable
service as long as it follows certain standards of communication
(interaction with other core network elements)

Asterisk is a free and open source application server that can be integrated
with your core. As to what services Asterisk offers: www.asterisk.org is the
best place to answer that.

And of course: in one network you can have multiple application servers
providing a rich gamut of features (Server A can offer voicemail services,
Server B can be a media server, Server C can be a session border control
[firewall type - preventing DOS and DDOS attacks]). The only common aspect:
they interact with the same single core (cscf).
