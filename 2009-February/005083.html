Hi Noriko,

> I'm testing to send messages with TCP, between UE,CSCF and AS.
> The problem is that message is forwarded to AS from SCSCF with UDP,
> in spite of the message transmitted with TCP from UE.
 
> How can I specify the protocol of messages forwarding AS ?

You cannot do that. It's up to the server to select the transport protocol
to be used for the next hop (one exception is sips-URIs which require TLS,
but this is not relevant for IMS).
