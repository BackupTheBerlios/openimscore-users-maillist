Hi Gray.

I'd see one problem. It seems that JAVA_HOME environment variable is
pointing to java executable. It should point to a directory. For example
in my system:

        edumoreno at walqa347:~$ echo $JAVA_HOME
        /usr/lib/jvm/java-6-sun/jre

Change the JAVA_HOME variable until you see:

        root at orion:/usr/lib/jvm/java-7-icedtea/jre/bin# echo $JAVA_HOME
        /usr/lib/jvm/java-7-icedtea/jre

By the other hand, FHoSS must be executed from deploy directory,
specifically from

        /opt/OpenIMSCore/FHoSS/deploy

So, you should see in your console something like this:

        root at orion:/opt/OpenIMSCore/FHoSS/deploy# ./startup.sh


Regards...
