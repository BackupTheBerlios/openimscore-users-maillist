Hi all!

We found following issue: 
  - Set up a trigger point for your AS to handle all REGISTER request
types (Reg, ReReg, DeReg)
  - Try to register -- you will receive 500 Internal server error

This error is caused by wrong XML schema in
modules/scscf/CxDataType_Rel7.xsd. tRegistrationType defines 3 element
while everywhere whithin the file where this type is used maxOccurs for
it is set to 2. So to fix this problem we should change maxOccurs from 2
to 3.
I attached an error log.

I'm sure it's easy-to-fix issue :)

Roman Sluchevskiy
