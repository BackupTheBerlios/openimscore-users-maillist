Hi Majdi.

The X-Lite configuration that I showed to you it isn't the best. It is
better to configure SIP Proxy instead Out Bound Proxy. So try with this
new configuration:

        Enabled: Yes
        Display Name:bob
        Username: bob
        Authorization User:bob at open-ims.test
        Password: **** (bob)
        Domain/Realm: open-ims.test
        SIP Proxy: pcscf.open-ims.test:4060
        Out Bound Proxy: <empty>
        Use Outbound Proxy: Default
        Send Internal IP:Default
        Register:default
        Voicemail SIP URL
        Forward SIP URL


Even better to put the IP instead pcscf.open-ims.test according to the
kind of network interface you are using (127.0.0.1 or public IP).

Using OutBound Proxy, your client will try to locate the registrar
through a DNS query from open-ims.test realm. Depending on DNS
configuration in client machine, it could not work.

Regards...
