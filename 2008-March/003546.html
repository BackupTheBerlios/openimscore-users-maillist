Hi Rifae,

> any one know the media encoding used in Open IMS Client (i.e : is it the
> AMR encoding or what), and if any one know any free JAVA package uses AMR
> encoding.

Open IC client uses:  	m=audio 8000 RTP/AVP 0 3 8
According to RFC 3551 table 4 this means PCMU, GSM, PCMA.

IMS communicator uses: m=audio 12000 RTP/AVP 0 3 4 5 6 8 15 18 
a=rtpmap:0 ULAW/8000 
a=rtpmap:3 gsm/8000 
a=rtpmap:4 g723/8000 
a=rtpmap:5 dvi/8000 
a=rtpmap:6 dvi/16000 
a=rtpmap:8 alaw/8000 
a=rtpmap:15 g728/8000 
a=rtpmap:18 g729/8000

I have no idea on a free AMR implementation.
