Erlend Trillerud wrote:
> hello again
> I know its not a openimscore issue but it I just wanted to be nice and 
> tell you how to solve this issue.
> After some weeks of testing different IBM J9 packages i finally 
> figured out how it works. Wasnt hard realy, just that the IBM user 
> guide kinda left out some key elements.
>
> -First of for the HTC tytn 2 you need to download and install the 
> ibm-weme-wm50-arm-ppro10_6.1.1.20061110-161633.exe on your computer
> -then find the zip file (standard located in) :  C:\Program Files 
> (x86)\IBM\WEME\runtimes\61\wm50-arm-ppro10\weme-wm50-arm-ppro10_6.1.1.20061110-161633.zip 
> and extract it.
> -Copy the folders to your mobile and the location is important or it 
> wont work: (root)\J9\PPRO10
> -Then run J9.exe in (root)\J9\PPRO10\bin\  then let it do its magic 
> wait for it to complete, the IBM install guid dosnt let you know that 
> you have to run it in the background when you install another program 
> so do that.
> -Let J9 run in the background. Then run the OpenIC lite cab file on 
> your mobile and install itl.
> -Run OpenIC lite and it should work.
>
> Tried it on a Qtek 9100 and it worked also. And with another HTC tytn 
> 2, but it dint work. SO I recomend to start fresh, backup your mobile 
> and do a software reset.
>
> -Erlend
