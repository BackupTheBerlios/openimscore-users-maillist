Hi All!
Last week I spend some time to improve OpenIMSCore image. I make some
improvements
1) New startup scripts
2) I Add SIPp to image
3) busy user. You make call to this user and SIPp script will always
answer "Busy here". Just start /opt/sipp/busy/busy.sh
4) No more 127.0.0.1. On every startup configuration files created
from templates (*.tpl) with real external IP.
5) Now by default you can connect to OpenIMSCore outside image. I
attach image with OpenIC_Lite configuration screen.
6) Image recompiled to support Pentuim3 PC's
7) Some config options (consoles, additional options) moved to
/etc/conf.d/openimscore

I also have plans to create some users like busy. One user must answer
with some voice message and hangup, another user must answer on
messages..

Please download and test this image, after testing I will fix all
problems, extract all scripts form image and send to Dragos for
official version.
To Dragos - is it possible to use macros in ser config files? Like
macros in SIPp files.

You can download image from
www.testik.com/OpenIMS/OpenIMSCore.2008-03-09.r538.rar (367 mb)
To use this image you can download free VMware Player -
http://vmware.com/download/player/
