good
> document for the same.  If anybody has the steps to integrate openims core

The "integration" is a trivial task. The application server has an IP
address and port where it expects SIP requests and responses to act upon.
You have to provide this address when you create the iFC in OpenIMSCore.
Details for that step can be found in http://www.openimscore.org/faq .
