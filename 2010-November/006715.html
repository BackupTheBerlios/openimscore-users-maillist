Hi Kapila,

> I have a query regarding the Mr interface of IMS. Is this interface only
> between S-CSCF anf MRFC. 

Yes, according to 3GPP TS 23.518.
The Mr reference point allows the S-CSCF to relay signalling messages
between an application server function and an MRFC

> Can an Application server directly communicate
> with MRFC for playing announcements over the Mr interface

Yes, but in this case a separate referece point has been defined: Cr.
See 3GPP TS 23.002 (6a.7.5a	Reference Point AS-MRFC)
