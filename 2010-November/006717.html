Hi Kapila

To complete Franz' answer, Mr' has been defined in the same reference for session control w/o the need to pass the S-CSCF.

While the Cr reference point shall be used to "provide remote prompts to the MRFC", Mr' is used to carry media control commands between AS and MRFC directly (3GPP TS 24.229). 
I would also interpret 3GPP TS 23.002 like this and say that Mr' is the pendant to Mr w/o S-CSCF; Cr is rather used for configuration of what might have to be expected by the MRFC over Mr' but not for the actual session control involving the MRFC itself.
