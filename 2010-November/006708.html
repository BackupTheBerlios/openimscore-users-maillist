Hello Thomas,

I am happy that you are one of the users of the emergency services in Open IMS Core.
I must admit that the README was old, I only updated the webpage of the emergency services:
http://openimscore.org/emergency 
with the installation guide at: http://openimscore.org/emergency_installation_guide

The reason you have problems is that the default PSAP parameters are not located in the E-CSCF, not in the LRF.
I have updated the README but I would recommend using the webpage as documentation.

The emergency branch was integrated in the trunk and from the last e-mails I think it is now stable. In the future the emergency branch will be closed.
If you have any other questions feel free to ask me, e.g. while installing the Postgres server for the LoST server there might be some issues as the documentation is written for an older version, 8.4 I think.

Best regards,
Ancuta
