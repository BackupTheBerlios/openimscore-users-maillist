> Is there any method present so that I can get register 200k when I am

You can do some tricks by modifying the scscf.cfg code at:

		if (!S_is_authorized("xxxxxx")) {
			S_challenge("xxxxxx");
			route(Service_Routes);
			t_reply("401", "Unauthorized - Challenging the UE");
			exit;
		}
		. . .

but  then you violate the IMS standards.
