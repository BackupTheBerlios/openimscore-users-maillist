Hi Gaurav,

> Just need to know if there is a way to get  a dump of SIP messages
> (received to, sent from IMS core) written to a text file ( format 
similar
> to what happens in Wireshark.)

The most comfortable way to dump the signalling messages is using tcpdump 
on
your Linux machine. The file which is produced can be read by Wireshark.
I typically use the command:
  tcpdump -i any -s 3000 -w /var/tmp/trace.pcap not port 22 and not port
3306
Via this command I exclude SSH and MySQL packets.
