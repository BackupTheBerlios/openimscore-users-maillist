How can we have "Diversion" in an Invite request through OpenIMS? Is

- A (1111) is calling B (2222)
- B is unreachable or not answering

attached as an AS in FOHSS

164.0.144.124 is IP of Voicemail server ( AS)

FROM = 2222
DIVERSION = 1111
