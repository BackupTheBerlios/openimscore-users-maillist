> I've just installed the virtual machine. I wanted to know which is the web
> address to access through other virtual machine. I mena, in OpenIMSCore VM
> is localhost:8080, but, what about from other VM?

If you change the numeric host address in
/opt/OpenIMSCore/FHoSS/deploy/hss.properties to a valid FQDN of the host you
will be able to access it from anywhere.
