Hi Franz,

It's useful information to me!
I have added this feature in OpenIMSCore, so my OpenIMSCore can
process the authentication of non-REGISTER request.

And I found a bug in code of the OpenIMSCore.
The function "find_credentials" in some usages are wrong, for example:

ret = find_credentials(msg, &realm, HDR_AUTHORIZATION_F, &h);

The correct code should be :

ret = find_credentials(msg, &realm, HDR_AUTHORIZATION_T, &h);

You can check the find_credentials() function to find the reason.
