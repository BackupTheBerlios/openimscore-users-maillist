Hi,
 
> i am not able to get rid of the 5060, is it possible to strip it off
> in icscf or force scscf to accept that invite?

It should not be that difficult to strip it off by using the function
subst_uri('/re/repl/flags') of the textops module in icscf.cfg. You have to
find the correct regular expression for this string-modification.
See the documentation in: http://www.iptel.org/ser/doc/modules/textops
