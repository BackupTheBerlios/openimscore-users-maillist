arjun,

you need to have IP connectivity between the machines. your second box will
act as SIP User End points. Ensure DNS is properly configured on the second
box (/etc/resolv.conf nameserver should point to the first linux box). The
SIP UE configuration can be found here:
http://www.openimscore.org/installation_guide#annexb

http://www.openimscore.org/installation_guide#step4 and
http://www.openimscore.org/installation_guide#annexc has all the details
clearly explained.

as the links suggest, try on loopback first then move on to the external
interfaces.
