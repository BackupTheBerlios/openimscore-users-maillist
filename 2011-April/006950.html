Irsalina,

> 1. Is there any way to retrive the information about what kind of terminal
> type that the user used when they're doing SIP registration or call
through
> openimscore? For example whether they use iPhone, iPad, tablette, etc?

There is the User-Agent header field usually provided in a REGISTER request.
But you have to be careful on relying on its content. It is up to the User
Agent to write anything into this header field.
Mobile operators usually do not rely on this header field but use the IMEI
(International Mobile Equipment Identity) which is provided during
attachment.

> 2. Could we find out in OpenIMSCore if an IMPU is doing multiple
registration
> in several SIP terminals and what header could distinguish them?

There is the function S_lookup in module registrar.c which provides the
Contact address(es) of  of an IMPU.
The different Contact addresses distinguish the different terminals,.
