Hi Muzammil,
For me it should work! May be you have a configuration problem buddy. In
your "scscf.cfg" try to find if you have something like :
modparam(“scscf”,”registration_default_algorithm”,”AKAv1-MD5″)
#modparam(“scscf”,”registration_default_algorithm”,”MD5″)
#modparam(“scscf”,”registration_default_algorithm”,"EARLY-IMS″)
#modparam(“scscf”,”registration_default_algorithm”,”HSS-Selected”)
modparam("scscf","registration_disable_early_ims",1)
modparam("scscf","registration_disable_nass_bundled",1)

I thing you have to play with those parameters especially this one :
modparam("scscf","registration_disable_early_ims",1).
try this modparam("scscf","registration_disable_early_ims",0). But as you
could see in the sccf configuration file they say that : " The next
authentication methods are implemented here but not yet completed in FHoSS"
so  "do not complain about bugs when using with FHoSS"
I hope that would help!
