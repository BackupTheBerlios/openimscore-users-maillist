If you look into the documentation of the function "P_assert_identity" in
module registration.c you will see what the function does:
"Asserts the P-Preferred-Identity if registered and inserts the
P-Asserted-Identity"

The problem is therefore the content of the "P-Preferred-Identity" header.
This identity is obviously not registered.
