That's true. After installation the two users alice and bob are already
provisioned if you have executed the mysql-script files (mysql -u root -p -h
localhost < xxxx.sql).

It seems that user bob is not provisioned because the UAA message shows the
error "DIAMETER_ERROR_USER_UNKNOWN". This means that either the Private User
Identity or the Public User Identity do not exist in the HSS.

Check the MySQL database (tables IMPI and IMPU in database hss_db) for that.
