On Wed, Feb 9, 2011 at 4:37 PM, tomas kovacik <kovacik at fiit.stuba.sk> wrote:

> dear OpenIMSCore users.
> i would like to introduce live CD of openimscore to the community. it has
> been prepared as a student's project at Slovak University of Technology
> with cooperation to ngnlab.eu project. it comes from original vmware
> image published at your page: http://www.openimscore.org/vm
> there has been configuration web page added which enables
> start|stop|restart of running core with possibility to change IP address of
> the core. configuration files of I-CSCF, S-CSCF, P-CSCF and others can be
> modified also.
> comming from our hard experiences, the advantage of such liveCD is that
> if a user wants to perform some basic tests, he does not need to setup his
> openims core but loads it from this live ISO which we provide.
>
> If you are interested in using it, you can download it at:
> http://147.175.103.82/~kovacikt/livecd/
