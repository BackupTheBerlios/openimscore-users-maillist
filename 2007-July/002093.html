Hi,
As per my understanding a contact can be updated using target refresh
request. And NOTIFY is one such request which is a target refresh.
Only RFC3261 talks about reINVITE as target refresh and no other states this

explictly but, there is a draft which talks about this
http://www.ietf.org/internet-drafts/draft-ietf-sipping-dialogusage-06.txt
(Refer section "5.4 Target Refresh request").
