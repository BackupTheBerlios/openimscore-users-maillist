Hei Darshak,

if you setup the ims core instances, you can go on with testing some
scenarios. get an ims client and register your user with this 
client to the domain, then you can instantiate a call between
a caller and callee. Afterwards you can try IFCs and so on.

This will also help you see if you have made any mistakes during
installation. But, always read the installation manual and faq's
carefully.

-Umut
