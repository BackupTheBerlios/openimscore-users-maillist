All the users in your domain will have the same host part in the IMPU.
However, sip:alice at open-ims.test can not use sip:bob at open-ims.test for
the obvious reason that they do not represent the same person. What you
want to is to have an implicit set of IMPUs that would be implicitly
registered when you register one of the members. This is called implicit
set and you have to also indicate in the HSS configuration that the 2
IMPUs belong in the same implicit set... what is still unclear?
