Hi Vinod,

The Operating System I'm using is Ubuntu.

I have encountered some problems during the mobicents installation :
- some are because I think mobicents was developped on Windows OS, and many
files are corrupted (run.sh, etc...) -> use a dos2unix program to avoid this
problem.
- the file run.sh is corrupted too (and I don't know why), but I've add a
change at the end of this file :

# Execute the JVM
   "$JAVA" $JAVA_OPTS \
      -Djava.endorsed.dirs="$JBOSS_ENDORSED_DIRS" \
      -classpath "$JBOSS_CLASSPATH" \
      org.jboss.Main -c all -b 172.20.107.179
#org.jboss.Main "$ARGS"

when you'll see the file, I think you understand why I've done this change

(I have took a long time in this error)

And it is not very easy to provision users in th mobicents server, I don't
know why...
If you choose this solution and have problem, contact me (even if I'm not
the rigth guy, some others may be very more capable with mobicents, I think
:)
