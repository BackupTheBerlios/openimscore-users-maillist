Yes. I just installed SEMS and tried a simple announcement. No problem.

I have attached:
- wireshark- and SIP-scenario-trace
- sems.conf
- ser.cfg (as1-sems.cfg)

Some installation hints:

Installation of SEMS
- wget ftp://ftp.iptel.org/pub/sems/sems-0.10.0-rc2.tar.gz
- tar -xvzf sems-0.10.0-rc2.tar.gz

- compile:
  - make all 
  - make install
 
Installation of SER 0.9.6
- wget http://ftp.iptel.org/pub/ser/0.9.6/src/ser-0.9.6_src.tar.gz
- tar -xvzf ser-0.9.6_src.tar.gz

- compile:
  - make all                           
  - make install

Good luck.
