Hei Darshak,

it's not sure that your setup is okay just because *.sh scripts
run and you see the debug consoles.

To my opinion your request stucks up somewhere in core and you get 504
from pcscf, we could see that if you send us a trace. you have to
capture the network traffic with wireshark, a tool which
does it for you. therefore you have to listen for certain
ports/protocols, like "sip","diameter" and "dns"(in this case
protocols). save it as pcap and send it here. checkout the wireshark
projects website how to install & how to use it

or do a simple ngrep (which i do usually) like
salvoxia> ngrep -d lo -W byline port 4060 or port 5060 or port 6060  
-assuming *cscf are running on the same machine

Umut
