Hello Raman,

For the same UE, you can't have 2 S-CSCFs at the same time. So Alice can
not register to both. But Alice can be registered to S-CSCF1 and Bob to
S-CSCF2.

What you need to do is to provision in the icscf database (sorry, no
GUI) all your S-CSCFs and then some capabilities for each S-CSCF.

Then in FHoSS you can assign a set of mandatory and optional
capabilities for each individual user. Then the I-CSCF would do
capabilities selection on REGISTER and redirect your user to the S-CSCF
that suits him the best (all mandatory capabilities supported and
maximum number of optional ones). If the first candidate S-CSCF does not
answer, the next ones would be tried.

The capabilities are just some integers that you should choose and
remember what each one means.

Oh, and remember that if there is already an S-CSCF assigned to a user
(you left the user in a registered state), any REGISTER requests would
be redirected there. To clear this association, just de-register.

-Dragos
