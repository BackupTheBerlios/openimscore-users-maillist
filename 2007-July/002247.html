He Memon,

Please take a look at scscf.cfg and change it so that you won't care
about what S_is_authorized() returns, for example.

Early-IMS is an authentication method where the IP address of the UE
should be stored on the HSS. The the S_is_authorized() will return true
if you enable early-ims (by default it is disabled, to reduce the
resource consumption when you don't use it) in the scscf.cfg and the
client tries to register from that IP address, or false if the IP
address is different. To close the full circle, your access network
should have a way of asserting that IP address for that client and
updating the value in the HSS - however, it would also work if you
update it by hand...

-Dragos
