Install mysql package (if u haven't installed it earlier) in /var/lib/mysql
where mysql-5.1.22-rc-linux-i686-glibc23/lib is the path of your mysql
library and write the following enteries in pcscf.sh

./ser_ims/trunk/cfg/killser pcscf
setkey -F
setkey -FP
LD_LIBRARY_PATH=/opt/OpenIMSCore/ser_ims/trunk/lib/cds
export
LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/var/lib/mysql/mysql-5.1.22-rc-linux-i686-g
libc23/lib
/opt/OpenIMSCore/ser_ims/trunk/ser -f /opt/OpenIMSCore/pcscf.cfg -D -D

Dushyant P S Dhalia
