If you mean to use Asterisk like a ISDN break-out gateway, the I-CSCF 
and S-CSCF configuration files already contain helping routing code - 
just look for "PSTN".

Now, for a pure and nice IMS-like architecture, you will need a proper 
BGCF and MGCF to do that. I think that Asterisk would do well as a MGCF, 
just not sure if it can do all of the BGCF. If it can't we could always 
have a bgcf.sh/cfg/etc example on how to adapt to Asterisk. If anyone 
wants to share this, we would happily include it in the project.

I don't have much integration experience with Asterisk, but we did quite 
a lot of integration with commercial alternatives. In most of the cases, 
such a BGCF adapter was required, just to smooth things up.... but from 
case to case, it did different operations.

Cheers,
-Dragos
