Hi Sergio!

You have developed all that stuff!! Congratulation! It is very powerful!
I've transformed my simple Asterisk PBX into a real Asterisk3G ...the
functionalities (especially Video IVR) are very innovative (for mute people
for instance...)

I think it could be very interesting to integrate Asterisk as an ISDN
breakout gateway! Do you have a clue on how to start?
The main issue here is that Asterisk does not support IMS SIP
signallisation.
I think we have 2 solutions:

1) modify the asterisk code directly...that's look diificult to do
2) develop a SIP to IMS (or IMS to SIP) gateway that could interact with
non-IMS servers.

Anyway, that project looks exciting (it would be great to have an open
source solution for 3G networks!!

Cheers,
Khaldon
