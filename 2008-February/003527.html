Hi Xuelin,

5.4.2 - these are the default credentials for the web interface and not
for MySQL

5.4.1 - these are the default MySQL credentials that FHoSS will use to
connect to MySQL if you look in hss_db.sql you will see at the end that
this credentials are added so that FHoSS can connect.

The credentials to actually create the databases by (the "mysql -u admin
-p < xxx.sql" commands) are the ones configured in your system when you
installed your MySQL server. admin is just an example of user with db
creation rights (usually this is "root").

Cheers,
-Dragos
