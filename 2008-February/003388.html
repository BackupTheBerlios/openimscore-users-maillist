Your client fails to calculate the MD5 response correctly.
DBG:S-CSCF:S_is_authorized: UE said: bddea74ec020a8c41f7fae69e968791b and we
expect 7e48f0f9db73af87ad975186727f87a9 ha1 48058be8b8f40fb9707c2679a0d12082

So I suggest that you update your client (IMS communicator). The latest
release can be downloaded from:
https://svn.berlios.de/svnroot/repos/imscommunicator/trunk/releases/

I remember a similar problem I had using the old version ("IMS-Communicator
070605"). I actually use "IMS-Communicator 071205" where above problem
disappeared.
