Hello Persa,

> I am trying to add an alias domain in Open IMS Core, but slightly stuck.

This cannot be done that easy. IMS is not a "simple SIP proxy".
IMS is based on a strict domain based network architecture (home network,
visited networks etc...). Also the database tables are structured according
to this architecture.

If you want use another domain you have to setup a second platform and 
enable roaming between the networks. But each network covers only a single
domain.
