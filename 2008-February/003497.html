for the usb-key - you have to be careful if you want to mount it in the
guest OS. For this to work, the key most not be claimed by the host OS
(or at least this is the case with linux, for windows as host it might
be easier, or?). Then all you need to do in vmware is to have the focus
of your virtual machine and normally the usb device is automatically
assigned to it - look in the vmware menu and if it is detected you will
see it there. I had good experience with host linux, guest windows and a
usb 2.0 webcam with windows-only drivers, but iTunes+iPhone fails...

I would rather use the Shared Folders feature from VMWare or network
access if you can, instead of trying this.

For the FHoSS console:

/opt/OpenIMSCore/fhoss_console.sh

it launches links2 in a fb graphical mode.

Cheers,
-Dragos
