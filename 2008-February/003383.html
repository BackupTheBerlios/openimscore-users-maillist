Hi Sabyasachi,
 
> I have tried with Authentication MD5 it works. Can you pl through some
> idea why Akav1-MD5 is not working?

The authentication methods between UE and HSS + S-CSCF have to be
consistent. Check the authentication method provisioned at the HSS for the
subscriber (Private User Identity) and compare it with the authentication
parameter in scscf.cfg (#modparam("scscf","registration_default_algorithm
...). Both have to be also consistent with the authentication method used at
the UE. - Maybe there is an inconsistence.

To be most flexible you can use "HSS-Selected" at the scscf.
