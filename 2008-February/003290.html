Hi Majdi.

Any Linux distribution where you can install the software required
should be suitable, but I can tell you that Ubuntu have all the packages
needed in their repositories.
You should install also the "bison" and "flex" packages or you'll find
errors during compilation.

The "Browser on the box..." is to say that the user management is made
through a web interface, so you'll need a browser in a machine that have
connectivity with the host where FHoSS node run.

Good luck...
