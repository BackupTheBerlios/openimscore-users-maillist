Hi.

The only environment setup you need is to set JAVA_HOME variable to root
directory of JRE. For example in my Ubuntu box I added this
to /etc/environment file (of course this need to be in sync with your
Java installation):

JAVA_HOME="/usr/lib/jvm/java-6-sun/jre"

All the software requeriments in the Installation Guide runs as default:
http://www.openimscore.org/installation_guide

You don't need a Tomcat Server. It is integrated inside FHoSS as a Java
threat.

Regards...
