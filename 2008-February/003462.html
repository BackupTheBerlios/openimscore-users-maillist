Just to have a clear picture on Open IMS Core:
Open IMS Core provides only the Core of an IMS platform as shown on the home
page http://www.openimscore.org/. A presence server is an application server
and is not part of Open IMS core. 

You can easily add an AS to the platform, but you need an AS from other
sources. The presence server of OpenSER might be an example.
