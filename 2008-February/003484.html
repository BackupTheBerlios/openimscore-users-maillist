Hi,

   The lines
   opt/OpenIMSCore/ser_ims/lib

   /xcap/parse_common_rules.h:30:27: error: libxml/parser.h: No such file or
directory
   /opt/OpenIMSCore/ser_ims/lib/xcap/parse_common_rules.h:31:25: error:
libxml/tree.h: No such file or directory
indicate that it is somehow not able to find the libxml/parser.h and
libxml/tree.h.
Check whether the file exists in the libxml2 folder and if they do then your
path is not set right in the makefile.
Make the changes to the include path and you should be able to solve the
problem.

Regards,
Sayaji
