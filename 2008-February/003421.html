There is no application server included with Open IMS. Open IMS includes the
following functional nodes: P-CSCF, I-CSCF, S-CSCF and HSS (FHoSS). This
might be your problem!

Maybe you got confused by the "Services" configuration options provided by
FHoSS (at somewhere like:
http://hss.open-ims.test:8080/hss.web.console/pages/services)? When you
"Create" an Application Server here then you just create configuration
details to be used in Initial Filter Criteria (iFC). The S-CSCF will route
requests based on the trigger points in iFC, and if there is a match (e.g.,
MSRP requests) then S-CSCF will route those to the AS pointed to in the iFC.

Still, you need to setup & install this application server. See
http://www.openimscore.org/faq/46 for more details on this. At
http://www.openimscore.org/links you'll find links to application servers
you can use.
