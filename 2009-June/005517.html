Hi Mauro,

Nothing to be sorry about, don't worry ;-). I find this discussion quite 
relevant for what happens with the project and the standardization.

Actually I am sorry, but I for one was not able to keep up in the last 
year or so with all the output from 3GPP. We are practically not finding 
the financing to follow the standardization work at least in its theory 
than if not in its implementation. And this is a pity, especially as we 
got some feedback that many people rely on it for doing interoperability 
tests. Also because it is open source, many see it as a reference for 
IMS, yet we can't keep up.

But as always, if someone is interested in this and would contribute in 
some way (financing, code, bug-fixes, feedback, etc), we don't have 
anything against it and well, it's open source and my organization does 
not "own" this project more than any other and everybody is free and 
welcome to improve it (there is no "fee", restriction or anything like 
that for becoming a project developer, we'd just like that you keep 
existing features present for others that might use them).

Cheers,
-Dragos
