Hi Franz,

With due respect I want to inform you that number can be added as
5550005 at 172.16.5.5 in FHoSS where 172.16.5.5 is the OpenIMSCore server's
IP address. For this to work I made some changes which are given below
cahnge in scscf.cfg
original>>
# Terminating
        if (uri=~"sip:(.*)@open-ims\.test(.*)" || uri=~"tel:.*"){
            if (S_term_not_registered())
                S_assign_server_unreg("open-ims.test", "term");
        }else{
            sl_send_reply("403","Forbidden - Dialog not found on S-CSCF or
Terminating user not suitable for unregistered services");
            exit();
        }

        route(Term);

        break;
    }
Changed>>
# Terminating
        if (uri=~"sip:(.*)@open-ims\.test(.*)" || uri=~"tel:.*" ||
uri=~"sip:(.*)@172.16.5.5(.*)"
){
            if (S_term_not_registered())
                S_assign_server_unreg("open-ims.test", "term");
        }else{
            sl_send_reply("403","Forbidden - Dialog not found on S-CSCF or
Terminating user not suitable for unregistered services");
            exit();
        }

        route(Term);

        break;
    }
After that call between the OpenIMScore numbers is okay.Am I right? I
wanted to change it because I wanted to interconnect OpenIMSCore with
SoftX3000 which uses IP address. So i wanted to use number in HSS as
5550005 at 172.16.5.5. Am I right.
