Hi,

you have some errors in your script-file.
Go back to SVN and look at the file "
http://svn.berlios.de/wsvn/openimscore/FHoSS/trunk/scripts/startup.sh"

#!/bin/bash
# --------------------------------------------------------------
# Include JAR Files
# --------------------------------------------------------------

echo "Building Classpath"
CLASSPATH=$CLASSPATH:log4j.properties:.
for i in lib/*.jar; do CLASSPATH="$i":"$CLASSPATH"; done
echo "Classpath is $CLASSPATH."

# --------------------------------------------------------------
# Start-up
# --------------------------------------------------------------

$JAVA_HOME/bin/java -cp $CLASSPATH de.fhg.fokus.hss.main.HSSContainer $1 $2
$3 $4 $5 $6 $7 $8 $9


The last command is start of HSS by calling the Java class.

If you have a truncated line starting with "cp $PATH" it is a copy command
and not a call of the Java runtime. Therefore you get the error.
