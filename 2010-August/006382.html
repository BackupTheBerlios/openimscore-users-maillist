Hi Sougata,

> In the mean time in the console of deploy where i am doing
> ./startup.sh, i am seeing that a error showing up " java.sql.SQLException:
> Access denied for user 'hss'@'scscf.open-ims.sougata' (using password:
> YES)".

The reason for this error is: you have changed the domain-name but you did
not adapt access permissions in MySQL database for the users of OpenIMSCore.
Look for the last line in hss_db.sql and issue an additional "grant" command
which reflects the actual domain-name.           
The same you should do for the "grant" commands contained in the last lines
in icscf.sql.

I hope that helps.
Franz
