> Hi Saleem,
>
> Just yesterday i solved the CLASSPATH issue.
> My java is installed in the /usr/lib/jvm/java-6-sun. After doing some
> google search i found out that in linux the java path doesn't get
> automatically updated in the bashrc file.
>
> So i just added 1 line at the end of the bashrc file by opening a new
> console and save it and close it.
>
> export JAVA_HOME=/usr/lib/jvm/java-6-sun
>
> My startup.sh file just looks like this , which i am running in
> FHoSS/deploy folder.
>
> #!/bin/bash
> # --------------------------------------------------------------
> # Include JAR Files
> # --------------------------------------------------------------
>
> echo "Building Classpath"
> CLASSPATH=$CLASSPATH:log4j.properties:.
> for i in lib/*.jar; do CLASSPATH="$i":"$CLASSPATH"; done
> echo "Classpath is $CLASSPATH."
>
> # --------------------------------------------------------------
> # Start-up
> # --------------------------------------------------------------
>
> $JAVA_HOME/bin/java -cp $CLASSPATH de.fhg.fokus.hss.main.HSSContainer $1 $2
> $3 $4 $5 $6 $7 $8 $9
>
> Hope it helps ,
