In IMS the correct registrar server is found by P-CSCF via DNS query based
on the domain name of the user. Therefore DNS has to be correctly setup.

> so what kind of configuration should I configure
> in registrar server's configuration ? the registrar server is SCSCF right?

What happens when you let the registrar server empty?

What happens when you set it to P-CSCF?

What happens when you set it to S-CSCF?

For all the cases above a tracefile created with tcpdump is necessary to
analyze what happens, e.g. via the command:
	tcpdump -i any -s 3000 -w /var/tmp/trace-xxx.pcap
