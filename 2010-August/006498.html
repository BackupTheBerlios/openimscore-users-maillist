> using Nokia N97, I successfully register on OSIMS ... but how to
> unregister the client? it is standardized that the client should send a
> Register with null period but does OSIMS enable to unregister from
> network?

Both methods are defined in IMS 
- deregistration (with expires=0)
- registration termination by the operator via RTR request from HSS to
S-CSCF

You can send the RTR via HSS GUI at Private User Identity - IMPI.
