The problem comes from changing the domain name but not in all directories.
When you use "configurator.sh" script you should apply it in the following
directories:
- /opt/OpenIMSCore/ser_ims/cfg
- /opt/OpenIMSCore/FHoss/deploy
- /opt/OpenIMSCore/FHoss/scripts

But be aware that applying the script for a second time requires some
modifications in the head of the script and because you did not change the
domain name in all cases it might be a mess.
I therefore propose you should take a fresh copy of OpenIMSCore.
