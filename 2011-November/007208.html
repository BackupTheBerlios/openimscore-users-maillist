> Can anyone help me out in the right direction?

OpenIMScore is not any of the above mentioned components. It is an open,
standards compliant IMS platform including an HSS.
You can e.g. attach application servers and use it for prototyping
development of new applications.

IMS is connected to a mobile network in an IP environment via Gi or SGi
reference point.
