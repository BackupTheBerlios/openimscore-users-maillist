Dear Eelco,

Configuring another destination realm was already supported in a pcscf 
parameter but the parameter was not used in the configuration file.

The key line was:

modparam("pcscf","pcc_dest_realm","my_realm")

Thank you for the feedback, now an appropriate line was added to the 
P-CSCF configuration file.

Best regards,
Ancuta Corici
