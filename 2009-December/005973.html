Hi Preeti,

regarding the version, you are right - this feature is not yet 
implemented. We're trying to cover as much functionality with as little 
effort. While the version would be helpful, this is only true on some 
limit situations. Anyway, patches and improvements are welcomed.

Regarding the storage of the reginfo document, the P-CSCF parses the 
information and keeps it's local version of the registrar synchronized 
with the status on the S-CSCFs. I don't see any need in keeping the 
entire document in memory after this update operation. Or is there any?

Cheers,
-Dragos
