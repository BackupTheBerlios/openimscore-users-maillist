Hi Doron,
 
> Q1. What is the way to access HSS at all from my Java Servlet or from Java
> in general ?

Use the Sh-interface for that (3GPP TS 29.328).


> Q2. What are the specific information I need to use (called AVP. Right? )

Use an UDR (User Data Request) command and ask for IMPU (IMS Public User
Identity).
