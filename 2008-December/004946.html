On Mon, Dec 15, 2008 at 4:29 PM, Eoghan Conway <econway at ait.ie> wrote:

> I've had a peek inside the sql files and it seems that the mysql
> permissions are being set for (userx)@localhost. The configurator script
> only replaces the ip address, so even after running it the db permissions
> will still be for local usage only. Your cscfs will be trying to connect
> from non-loopback addresses. Try replacing the @localhost with @192.168.x.x
> before reloading the sql files.
