Hi Noriko,

could you please start a trace on the client and send it to us?

I tested it and it worked... All that  was required was that the 
pcscf.open-ims.test entry in DNS was a AAAA one.  Then the client will 
try to make a TCP connection to that address, just to discover what its 
outbound IP address is. Do you see this TCP SYN? And in that java 
implementation there is no difference between the IPv4 and IPv6...
