Hello Prateek,

The responses are sent to the port_uc - User Client. When the Endpoint
is doing UAC, then it sends requests from port_uc and should be ready to
receive answers for those requests on the same port. Only requests are
sent towards port_us.

Then about your error, please register the same contacts that you are
going to use: you registered 192.168.1.2:5060 and then you use
192.168.1.2:1052 because of IPSec. So the error below was normal.

-Dragos
