Hi Samir Patry,

You probably didn't configured well your mysql. Please use one of the
many tutorials on the web to configure it properly.

And please be more specific when something is not working...we cannot
guess everything, as could be many reasons "why the HSS cannot
communicate with the mysql"...
Can you execute mysql commands from console? the manipulation of the
tables is working properly? did you added successfully the users from
the hssdb.sql script etc...please give us details in what step you are
currently.

If your mysql is configured properly and the hibernate.properties
configuration file weren't modified, the HSS should connect successfully
to the mysql server. Please indicate if you made some changes...probably
something is not configured well.
