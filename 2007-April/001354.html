Samir,

Are you correctly following the instructions on 
http://www.openimscore.org/docs/install.html#step4 ?

The dump.sql line is just an example to tell you how to import a file in 
mysql. Dump.sql shouldnt really exist on your OpenIMSCore installation, 
and the contents of your file seem to be more like an XML configuration 
for a Diameter Peer rather than SQL instructions to create tables and 
insert the data in your mysql database!

The files you should import are files like hssdb.sql and icscf.sql. You 
should find lines in them such as 'CREATE DATABASE', 'CREATE TABLE' etc.

Make sure you have the rights to create databases on your mysql 
installation because if the first step (to create a database) fails the 
rest of the instructions will fail too. I think mysql needs to be at least 
version 4.
