I got it...

the problem was I don't use NAT.
So, the pcscf crashed every time the function was called...

I've changed on the pcscf.cfg this line :
modparam("pcscf","NAT_enable", 1)

to :
modparam("pcscf","NAT_enable", 0)

so, it works...

When not using NAT, I think this line have to be like that...
Maybe I'm wrong, but I've tested, and it works... ;)
