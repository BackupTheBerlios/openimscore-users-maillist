Hi!
Yes, actually we managed to use the Cisco 7940 sip phone with the Open IMS
Core :)
Of curse you have to use the SIP2IMS gateway...
Following, the configuration we use (change according to your domain):
 
3. Network Configuration
  3.14 DNS Server 1: 10.112.64.12 (which resolves pcscf.ptinovacao.pt)
 
4. SIP Configuration
  4.7  Messages URI: ptinovacao.pt
  4.22 Outbound Proxy: 10.112.64.12 (pcscf.ptinovacao.pt)
  4.23 Outboudn Proxy Port: 3060 (SIP2IMS gw)
  4.1 Line 1
    4.1.3 Authentication Name: bob at ptinovacao.pt (private user id)
    4.1.6 Proxy Address: ptinovacao.pt
    4.1.7 Proxy Port: 5060
 
Hope you find this useful..
If you managed to get it work with other configurations, please let us know
