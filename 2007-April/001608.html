Hello,
 
As some of you may already be aware, PT-Inovação has released and is supporting the IMS-Communicator, an open-source project based on an older version of the java softphone SIP-Communicator, which implements many IMS procedures and is interoperable with the Open IMS Core.
 
To get the source code, binary releases, documentation and support, please refer to http://imscommunicator.berlios.de.
