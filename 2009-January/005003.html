Hi Srikanth,

> Meanwhile Is there any other mechanism to collect the trace from 
> OpenIMS?

Yes. Use tcpdump at the Linux core node.
The simple command ti create a tracefile which is compatible with
Wireshark (pcap-format) is:
tcpdump -i any -s 3000 -w /var/tmp/trace1.pcap

If you want to avoid e.g. SSH and MySQL packets as in my case because I
am working with VMware, you can use tcpdump -i any -s 3000 -w
/var/tmp/trace1.pcap not port 22 and not port 3306
