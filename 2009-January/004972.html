Hi Suresh,

One of the main IMS concept is the user based filtering and triggering
of signalling. So all you need to do is configure an Initial Filter
Criteria in the HSS, assigned to a user and make it so that it would
filter your INVITEs for example and forward them to your call control AS.

Modifying the scscf.cfg for the purpose of triggering Application
Servers would be wrong as you would hard-code the signalling path.

Cheers,
-Dragos
