True, one more SIP server is running on the 5060 port. That's why I had
to run the i-cscf on port 6050.

And to setup the DNS, I just followed the instructions from OpenIMS
installation guide:
1. Copy the file /opt/OpenIMSCore/ser_ims/cfg/open-ims.dnszone to where
you configure it above (pri/open-ims.dnszone)
2. cp /opt/OpenIMSCore/ser_ims/cfg/open-ims.dnszone /var/bind/pri/
3. chown -R named:named /var/bind/pri/open-ims.dnszone
4. /etc/init.d/named restart

Also I changed the /etc/named.conf to add this zone.
