Hi Josef,

by default, out of the box, the Open IMS Core comes with an example 
Service Profile, containing only one Initial Filter Criteria, that 
matches exactly for presence, so that should be fine. If you changed the 
configuration or still have issues, you should send a trace showing the 
messages and the flows between the Open IMS Core components.

For the 2nd issue, the error seems to be quite self-explanatory - you 
can't unpublish while you are not registered. Actually you can't do 
anything in IMS with a client while you are not registered.

Cheers,
-Dragos
