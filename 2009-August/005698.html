Hi all,

I have just uploaded a newer OpenIMSCore in a bottle image: 
http://www.openimscore.org/vm .

New things:
- yes... I for one gave up on Gentoo, it's Ubuntu 9.04 now. Hopefully 
this will be easier to update and live with than the old Gentoo
- MONSTER - read this before downloading: 
http://www.monster-the-client.org/monster-license
- parallel IPv4 and IPv6 installations
- services to start all as well as lots of scripts to attach to 
components running in screens, start/stop/restart or update. See what is 
in /opt/ for more.
- newer OpenIMSCore in side (r657). Not the latest though as we're just 
doing some major merges and changes on the trunk and that might not be 
so stable.
- double in size ~1GB. If anyone knows how to reduce this further, 
please let me know.


Any feedback is appreciated.

Cheers,
-Dragos
