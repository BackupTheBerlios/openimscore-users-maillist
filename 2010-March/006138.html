Hi,

i suspect your IP phones do not support the default authentication  
algorithm for IMS (AKAv1-MD5).
You can change the default algorithm in the "scscf.cfg" file. Search  
for a line containing
"modparam("scscf", "registration_default_algorithm", "AKAv1-MD5")".  
There are several commented lines below allowing you to enable other  
authentication algorithms. Try and see if one of them works for you.  
You should always have only one line active.
If the line "modparam("scscf", "registration_default_algorithm",  
"HSS-Selected")" is active you can change the authentication algorithm  
in your HSS web frontend.

BTW, you should add wireshark traces to your messages when asking for  
help, this makes it a lot easier to debug problems.
