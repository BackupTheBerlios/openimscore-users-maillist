Hi,

seems your user is not correctly provisioned. Do you use a  
preconfigured user or did you create one yourself?

I have one idea that you could try:
- go to the FHoSS web interface
- in "User Identities", "Public User Identity" search for your user's  
name, e.g. "bob"
- check that <your domain>, e.g. "open-ims.test" is listed under "List  
of Visited Networks"
   - if that entry is not there you can add it with the field above,  
"Add Visited-Networks", your
     SIP domain should be in the drop-down list.


Hope that helps, regards
