Hi,

HSS users are defined in "<FHoSS_folder>/deploy/conf/tomcat-users.xml",
e.g. "/opt/OpenIMSCore/FHoSS/deploy/conf/tomcat-users.xml" if you  
followed the installation guide.

default users are:
   "hss" with password "hss" for a read-only user
   "hssAdmin" with password "hss" for a read/write user.
