I guess you had some trouble when you downloaded the source code. There
are some files that are common which are kept as link in svn and appear
in multiple modules. A clean "svn checkout" should help you get the
proper directory structure.

Also, better to checkout using linux svn client since Windows SVN client
( TortoiseSVN for example) can not resolve symbolic links.
 
HTH... Sachin
