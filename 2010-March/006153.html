Hi Rebeca,

unfortunately, we can only provide our best-effort/free support for the 
latest revision of the project. So I can not help you with an older 
revision. Is there a blocking reason that you are not upgrading?

The latest revision was recently revised for the purpose of supporting 
the Rx interface properly, so that integration with an EPC system is 
possible. In principle, the code should be general enough to also 
support the Gq/Gq' interfaces.

For downloading the OpenIMSCore code, building and using it, please see 
our installation guide: http://www.openimscore.org/installation_guide

Cheers,
-Dragos
