Hi manal, the xml file is where you refer your video file and media server. I think you have to first configure a media server and store video files before using the IPTV server. Please look at the following page.
http://uctimsclient.berlios.de/uctiptv_howto.html

You can use Darwin Media Server as a media server. Then you have to specify the media server (not the iptv server) together with the video file you want to associate with each channel in your xml file.

Hope this will help!
