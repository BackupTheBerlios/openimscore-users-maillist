Hi Claudio,

here are my scripts that I use to easily set-up a core. You still need 
to checkout the ser_ims and FHoSS into /opt/OpenIMSCore, but after that 
you have all the required init/update/reconfigure/start/stop/attach 
scripts inside the bin directory and all the configuration into etc.

Of course, I use screen and there are init.d scripts included. The setup 
is dual, with one IPv4 and one IPv6 environment.

I plan on creating soon a new Ubuntu-based VM image and these scripts 
would make it directly there. So any feedback would be appreciated.

Cheers,
-Dragos
