> Hi Sanjoy,
> The problem that you are facing is because you are running the
> "icscf.sh" file from /opt/OpenIMSCore which I understand,is a
> pre-requisite but after you run the script file, a lot of processes
> are being generated which need to be killed before your icscf comes up
> properly. And these processes are killed by the "killser" command. So,
> the location of "killser" has to be changed in your "icscf.sh" file.
> Open "/opt/OpenIMSCore/icscf.sh" file and duly set the apt path to
> your killser,that is change "killser icsf" to "./ser_ims/cfg/killser
> icscf". Hopefully,this should make your icscf run.
>
> Regards,
> Pinak
