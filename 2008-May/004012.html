Hi all,

Configuring a PSI you should "unclick" the "can register" checkbox in
the IMPU, but when you click in "save" it appears "selected" again, as
you say Persa. 

I had the same problem, but it's not important. You can change the "can
register" value in the IMPU table in the database directly with MySQL
Query Browser but again it's not necessary. I'm using PSI with "can
register" value equal to one (selected) and it works perfectly. (I guess
this value is only checked when you are going to register an IMPU, but
is not if you are not)

Regards,

Enrique
