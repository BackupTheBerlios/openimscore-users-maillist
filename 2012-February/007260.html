Hello Andrés,

Unfortunately, the deb packages provided on the UCT IMS client website 
are obsolete and they won't work with the latest Ubuntu version.

Try compiling the client from the source code. Before you do that you 
have to install development packages (those ending with -dev) for all 
the dependencies of UCT IMS client. From what I remember you may expect 
some problems with libvlc2, the UCT IMS client uses some old API which 
is no longer available in the UCT IMS client. In order to compile it I 
had to modify the source code of the UCT IMS client.

If I find my modifications I may send them to you. Anyway, the UCT IMS 
client project doesn't seem to be very mature. From what I noticed the 
project is no longer updated and the client itself could be more a 
testbed than any professional app.
