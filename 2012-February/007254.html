Hi Peeyush,

> I checked every module to find something about clf.open-ims.test, however
> I was not able to figure out what is its purpose and how can I configure
it.

The CLF(Connectivity Location Function)  is defined in ETSI ES 282 004.
P-CSCF communicates with CLF via e2-interface.
You should deactivate e2-interface in P-CSCF. This should be disabled per
default (see below).

# e2 Interface configuration (NASS-Bundled Authentication)
#modparam("pcscf","forced_clf_peer","clf.net1.test")
modparam("pcscf","use_e2",0)
#modparam("pcscf","use_e2",1
