Your problem might be that you did not apply the script to all relevant
directories containing configuration data (e.g. cfg- or  xml-files).
I apply the script to the directories
- /opt/OpenIMSCore/ser_ims/cfg
- /opt/OpenIMSCore/FHoss/deploy
- /opt/OpenIMSCore/FHoss/scripts

Be aware that the script is designed per default to be applied only once. If
you want to apply it a second time you have to modify some variables in the
header of the script. Take a look.
