Yes it is. Why? Because we have tried to make the whole installation
process very easy to understand an start with, the initial configuration
assumes a local-loopback-core only. If you wish to use clients,
application servers and any other nodes from outside you will need to
reconfigure the core for your particular IP and DNS requirements.

-Dragos
