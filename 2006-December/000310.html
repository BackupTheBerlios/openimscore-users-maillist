Hi Ben,
Follow these steps to add a user ben at open-ims.test
1. Go to /opt/OpenIMSCore/ser_ims/utils/gen_ha1/
2. generate ha1b ./gen_ha1 ben  open-ims.test  ben
adebaa5b679f75d7e19b3a7a118f35e0
3. Add user ben at open-ims.test in sip2ims.sql similar to bob/alice. Use the
ha1b generated above for the new user.
3. Go to /opt/OpenIMSCore/FHoSS/scripts/userdata.sql
4. Add the new user similar to alice/bob.
5. Update MySql with the new userdata.sql and sip2ims.sql
