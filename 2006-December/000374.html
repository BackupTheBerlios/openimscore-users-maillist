Summary: sip2ims_via setting

Details: If the sip2ims gateway is running on port 5060, the port number is not written into the via header.

The sip2ims_via setting must have "0" as port numver for a correct identification of the sip2ims via header.

Example: modparam("pcscf","sip2ims_via","1.2.3.4:0")

"0" is no valid port number (IANA: "Reserved")
