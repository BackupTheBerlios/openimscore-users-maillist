Hi,

We have tested with eyebeam and we where able to make calls and
terminate them correctly, but had to patch the P-cscf like Dragos says
on the following thread:
https://lists.berlios.de/pipermail/openimscore-users/2006-November/000124.html

We also decided to change the Route block on our sip2ims script to the
following:

route{

        route(Sanity_Checks);

        if (method==REGISTER){
                route(REGISTER);
        }
        if(src_ip!="10.112.64.12" ){<---- this is the IP of the P-CSCF
                route(PCSCF);
        }

        if (method=="INVITE" ||method=="SUBCRIBE"){
                record_route();
         }

         loose_route();

         if (!t_relay()) {
           sl_reply_error();
         }
    exit;

}
