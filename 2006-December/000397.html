Hello Gaurav,

> ./icscf.sh: line 3: killser: command not found
This is problem caused by faulty configuration of the killer script.
You can find this script at /opt/OpenIMSCore/ser_ims/cfg.
Copy the script at the location where you have the icscf.sh script and set
it executable.

>  0(20463) new_connection: Can't connect to local MySQL server through
> socket '/var/lib/mysql/mysql.sock' (2)
There is a problem with your mysql-installation. 
Did you install it correctly?
Does mysql run?
Can you call mysql in command-line mode?
