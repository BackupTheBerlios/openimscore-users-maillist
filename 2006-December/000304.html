Hello,
Please check your settings for alice on KPHONE
Full name: alice
User part of SIP URL: alice
Host part of sip URL: open-ims.test
Outbound Proxy: The IP adress of server where SIP2IMS is running and the
port no:3060 ( e.g 10.10.200.216:3060)
Authentication username: alice at open-ims.test.
 
Important: you  must run SIP2IMS such that it listens on the network IP
( eth0 , and not local port as in default configuration)
What i mean is, your SIP2IMS should not be running on 127.0.0.1:3060 but
on your eth0 IP ( for eg. 10.10.200.216:3060). For this you have to
modify the sip2ims.cfg file.
