Carlos,

In my opinion your problem lies more in the mySQL privileges area than in the FHoSS itself.

Please check the privileges allowed for the user hss in the relevant mysql tables:
user
db
host

In my opinion, you have privileges set for hss only for connecting from "localhost".

Plus, check your my.cnf for the bind-address. This specifies what address your mysql is listening on.

Again, this is a question more related to mySQL than to HSS.
