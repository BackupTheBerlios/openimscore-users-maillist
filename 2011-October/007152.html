Hi Isshed,

> I have installed 2 openIMS on 2 different linux machines. I have 2
endpoints
> registered on them i.e.  one on each openIMS. Is there any way I can make
a
> call between these 2 endpoints?
> How to configure the openIMS to communicate on each other.

There are 2 preconditions:
1) Both domain names must be resolvable pointing to the I-CSCFs of the
respective domains
2) Roaming has to be enabled per PUID and network

Then it should work.
