>If you are using SIP Client ( Non IMS client ) make default Digest Algo to
MD5

Edit /opt/OpenIMSCore/scscf.cfg

uncomment MD5

and comment other two

#modparam("scscf","registration_default_algorithm","AKAv1-MD5")

#modparam("scscf","registration_default_algorithm","AKAv2-MD5")

modparam("scscf","registration_default_algorithm","MD5")



>IMS clients uses AKA-V1-MD5
