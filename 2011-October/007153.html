hi isshed.
the most problematic thing on this is DNS configuration. my setup is as
follows:
- each IMS domain has its own DNS sserver which resolves name requests for
domain inner nodes
- the third  DNS server is set as forwarder of IMS domains DNSs. it resolves
ICSCF of each IMS domain as its entry point for requests originated from
other domains

such setup ensures that outside nodes do not resolve i.e. scscf.domain1 by
its IP address but all requests are routed through I-CSCF.

if you need more help, i could provide my BIND9 config files.
