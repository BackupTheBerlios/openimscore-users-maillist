Hi Tran,

> I want to build a application server which connects to OpenIMSCore.
> I use Asterisk as application server.
> But when OpenIMSCore connects to Asterisk with message REGISTER, Asterisk
> replies message 401 : Unauthorized. Consequently, the call is terminated
> How do I config to connect between OpenIMSCore and Asterisk ?

The question is not very precise. Asterisk and OpenIMSCore are two different
systems. I guess you want to re-use an application of Asterisk in
OpenIMScore.

In this case you have to select the application which you want to reuse and
get a clear view how the application should be invoked via iFCs. Than
analyse how Asterisk is doing this and try to configure your environment
accordingly.

But be aware: there is no one-to fit-all solution. You have to first decide
on the application you want to offer.
