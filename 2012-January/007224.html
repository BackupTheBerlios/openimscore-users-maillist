> changed the ip in pcscf.cfg and pcscg.xml.
> kindly tell me what els file i have to reconfigure,????

You should use the configurator.sh script.
It has to be applied to all relevant script and configuration files containing IP address and domain.
Apply the script to the following directories:
- /opt/OpenIMSCore/ser_ims/cfg
- /opt/OpenIMSCore/FHoSS/deploy
- /opt/OpenIMSCore/FHoSS/scripts

Be aware that the script has some default parameter set in its header part, which you have to modify if you want to apply it a second time with different environment.

That should help.
