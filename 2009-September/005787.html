Hi Ghaith,

Unfortunately there are some changes that Kamailio and OpenSIPS would 
need in their core to support the cdp module. SER (iptel.org) has these 
and also does the new merge between SER&Kamailio (sip-router.org).

I have never heard about anybody that used OpenSIPS with it.

But on Kamailio there were some developers which unfortunately did not 
contribute back. This does not matter so much considering the new 
SIP-Router merged project, which should work with the latest changes in cdp.

The documentation can be built from the source code by using doxygen.

Cheers,
-Dragos
