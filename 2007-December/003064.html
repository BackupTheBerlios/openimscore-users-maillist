Hi,

IPSec is by default not mandatory, so there should be no need to disable it...
If you feel better by disabling it modify in pcscf.cfg
  modparam("pcscf","use_ipsec",1)
to
  modparam("pcscf","use_ipsec",0)

...btw, I think the problem with IPSec you suggest would cause 494 "Security Agreement Required"...

br, Klemens
