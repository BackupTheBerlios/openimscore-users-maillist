Hi Jorge,

> Does the Core have any tool to make that happen?

There are of course several possibilities to do that. For highest
flexibility you should use an Application Server, but I am sure there are
also more simple possibilities using existing SER module functions. 

First you have to decide if S-CSCF should do that filtering or the P-CSCF.
Than you have to modify (include some filtering code) in the corresponding
cfg-file. That should not be that hard to do (look e.g. at the functions of
textops module).
