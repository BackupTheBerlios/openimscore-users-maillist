Looks like your client uses a NAT-helping functionality that replaces
it's contact with the discovered NAT address. Disable this in the client
and it should work.

To make it easier for our users we preferred to have "server-side" NAT
helping functionality. Of course, you can always change this from the
P-CSCF source code if you would like - in the
cscf_get_originating_contact() function.
