    The first step is to include the address of the SIP-server (P-CSCF) as
    option 120 in your DHCP server. Then check if the option 120 with the
    correct address is received by the client.

    Then, if the client honours the option 120 is another issue. Maybe there are
    some clients which comply with RFC 3361 ...

    Keep us informed if you have success.
