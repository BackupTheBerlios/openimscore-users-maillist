Dear Kedy,

 

Looks like you are using the wrong Java runtime package and we have
never used the client together with the emulator. The Java profile
specified is the personal profile, from the screenshots I see it is
trying to run it as a MIDLet….

 

For your physical device, do the following;

    * IBM Registration
      (https://www14.software.ibm.com/webapp/iwm/web/preLogin.do?source=weme&S_TACT=104CBW71&S_PKG=dl61_win&submit.x=54&submit.y=8
      <https://www14.software.ibm.com/webapp/iwm/web/preLogin.do?source=weme&S_TACT=104CBW71&S_PKG=dl61_win&submit.x=54&submit.y=8>
      )
    * Downloading J9
      http://www14.software.ibm.com/download/data/web/en_US/trialprograms/N882497S32820G44.html?S_TACT=104CBW71

    * Select “Windows, Windows CE, Windows Pocket PC, Windows XP:…”
      option and click on continue
    * On the next page select the executable and installation
      instruction for CDC 1.0/Foundation 1.0/Personal Profile1.0 for
      Windows XP/X86 and install J9 on your device
    * After installing j9, install OpenIC_Lite “directly” as described
      on the accompanying installation guide.

 

If you still encounter problems, let me know.

 

 

Best Regards

 

- Alice Motanga -
