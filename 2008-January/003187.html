Hi Persa,

Well, congratulations ;-). We hope that it is easy to use.

you actually need to modify the scripts. But... We always built it with
authentication in mind, so there are some nasty implication if you do
so. I would suggest something else:
- use MD5 - we do support it and all clients support it
- use Early-IMS and configure in the HSS the IP address with the address
of the client machine. Then you will get no more 401 if you send a
REGISTER that matches that IP address. From the client's perspective
it's like there was no authentication.

Cheers,
-Dragos
