> The following is my steps:
> 1.modified /etc/bind/db.open-ims.test
>   changed all 127.0.0.1 to MYIP
> 
> 2.modified all .cfg .xml and hss.properties with configurator.sh
>   changed all 127.0.0.1 to MYIP

That is not all places where the IP has to be changed.
There are several directories containing IP-address specific content.
I apply the script configurator.sh to the directories:
- /opt/OpenIMSCore/ser_ims/cfg
- /opt/OpenIMSCore/FHoss/deploy
- /opt/OpenIMSCore/FHoss/scripts

Be aware the script per default works for origin IP = 127.0.0.1.
If you apply it a second time you may have to change the origin IP in the
within the script. Look into the header of the script.
