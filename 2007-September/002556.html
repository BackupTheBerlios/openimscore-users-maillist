Hello,

> Could any one tell me what should be the command I use to start
> scscf.sh scscf.sh pcscf in parallel :
> 
> If I install all the components in same host, what should be the order
> to start ?

That's not a big problem. I start all in the following sequence:
- hss
- p-cscf
- i-cxscf
- s-cscf

But the implementation is robust. Hss and i-cscf and s-cscf need each other
for the diameter connection. If any one is not available you get repeatedly
an error until the peer is available.
