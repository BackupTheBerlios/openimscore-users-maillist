"Peer scscf.open-ims.test can not be resolved" implies that your DNS
server is clearly not working and you can test it for yourself by
running this command:

$dig @127.0.0.1 scscf.open-ims.test

You need to ensure that you have followed the DNS part of step 4 of
the installation process;
http://openimscore.org/installation_guide#step4

and there is an FAQ about the DNS server here too:
http://openimscore.org/installation_guide#annexa
