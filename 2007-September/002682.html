Hello Santosh,

The correct standardized string for aes-cbc in the Security header is
actually "aes-cbc" and not the one that you mentioned. Then this will
translate to "rijndael-cbc" with the linux 2.6 IPSec implementation.
Please see the ser_ims/modules/pcscf/security.c file.

-Dragos
