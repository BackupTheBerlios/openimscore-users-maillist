Hi
 
If you are no longer using the default ip address (i.e 127.0.0.1),
you'll have to edit /etc/hosts by adding these lines:
 
  10.21.21.89 open-ims.test
  10.21.21.89 pcscf.open-ims.test
  10.21.21.89 scscf.open-ims.test
  10.21.21.89 icscf.open-ims.test
  10.21.21.89 hss.open-ims.test
 
Thereafter, you can try to ping them and see if you are still getting
the same error. That's how I solved that problem.
