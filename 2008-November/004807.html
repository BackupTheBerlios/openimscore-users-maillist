Hi Mosbah

The line "ERROR 1045 (28000): Access denied for user 'root'@'localhost'
(using password: YES)" means that your mysql password was rejected - have
you made sure that you are using the correct password ? Perhaps try log in
using a terminal and the command " mysql -u root -p ". It should ask you to
enter a password, after which you should see something like

"Welcome to the MySQL monitor.  Commands end with ; or \g."

Hope that helps
