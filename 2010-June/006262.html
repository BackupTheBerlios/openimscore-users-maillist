Hello Antonio,

By reading the error output I think there is no file at the:
/usr/lib/ser/modules/emerg_info.xml

By default (if you have checked out openimscore at /opt/OpenIMSCore/ser_ims)
this file is at /opt/OpenIMSCore/ser_ims/modules/pcscf/emerg_info.xml
see http://svn.berlios.de/viewcvs/openimscore/ser_ims/trunk/modules/pcscf/

as in the default configuration file, the setting for the emergency numbers are :

modparam("pcscf","emerg_numbers_file", "/opt/OpenIMSCore/ser_ims/modules/pcscf/emerg_info.xml")


Please check what path have you configured and/or copy the file at the desired location and try again.

Best regards,
Ancuta
