Hi Jamal,

These application servers are available, but not for free. Please
understand that we actually had to/have to finance the Open Source IMS
Core from somewhere. However, we are open towards any proposals and we
could even share the whole environments for development purposes.

SIPSEE - there was a typo there - SIPSEE comes from SIP Servlet
Execution Environment (and not SIP Server EE as I said before).

SIPSEE complies with the SIP servlet API as definied in JSR 116. More
information can be found here:

http://www.fokus.fraunhofer.de/bereichsseiten/testbeds/ims_playground/playground/as_servlets.php?lang=en
http://www.fokus.fraunhofer.de/bereichsseiten/testbeds/ims_playground/playground/application_Server.php?lang=en

-Dragos
