Hello Bindu,

The secret-key is a 16 bytes long binary password. If you would use a
text (as most clients ask for it in a text box), it would be truncated
to the first 16 chars or padded with zero characters until 16 bytes.

For gen_ha1 - you are right, sorry for the mistake - fixed now. The
gen_ha1 utility can be found in ser_ims/utils/gen_ha1/ (you need to do a
make in there to build it).

-Dragos
