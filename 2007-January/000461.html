Details: UDR failes in some cases, when it should return a Success response. 
The suggested test for repository data was done:

1. UDR for non existing repository data 
2. PUR for non existing repository data 
3. UDR for existing repository data 
4. UDR for the same existing repository data 
5. UDR for the same existing repository data
