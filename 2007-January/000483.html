Hei Sudhakar,

you have to `make` the ser which means creating your hardware dependent shared libraries
and executables on the machine that you want to install to. There are 
no binary packages that is maintained.

ser_ims wouldn't work unless you dont have a HSS(u can install FHoSS for that reason). So as far as i 
understand your question, yes , ser_ims source is a part of OpenIMSCore project.

Hope this helps,
