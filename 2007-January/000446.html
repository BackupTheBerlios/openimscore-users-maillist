Hi Jose,

Add more shared memory to the CSCFs using the command line parameter and
then increase the sizes of the hash-tables. Also tune the number of
forked processes based on your hardware configuration: too few and it
will just not load the CPUs, too many and it will spend to much time
while context switching.
