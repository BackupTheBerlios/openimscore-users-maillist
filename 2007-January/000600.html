So the IMS core (without the SIP2IMS) now supports MD5 too.

Now there are 2 ways of doing that:
- simplest way - from the client put a algorithm=MD5 in your initial
Authorization header
- other way - change the default authorization method in scscf.cfg from
AKA to MD5
