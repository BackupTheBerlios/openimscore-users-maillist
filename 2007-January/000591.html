The output gives you a hint....there's an error in the SIP headers,


>   parse_contacts(): Error while parsing params
>   2(3997) contact_parser(): Error while parsing contacts
>   2(3997) parse_contact(): Error while parsing
>   2(3997) ERR:I-CSCF:cscf_parse_contacts: error parsing contacts
> [<sip:alice at 127.0.0.1:5061>;]
                                                 ^^

concretely you should remove ; from the contact header in the sipp file

>          Contact: <sip:alice@[local_ip]:[local_port]>;

should be:

Contact: <sip:alice@[local_ip]:[local_port]>
                                                                    ^^

Hope it helps,
