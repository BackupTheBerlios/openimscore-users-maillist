> I read your old email and discussion on OpenIMSCore Mailing List. I want
to
> established IMS and PSTN connection. I use PSTN gateway, Huawei U-SYS
> IAD104H.
> I want to register the PSTN Gateway as a User in IMS. But, after I
configure
> the PSTN gateway, PSTN gateway need to register to the IMS. I try register
> the PSTN gateway as IMS User,but the registering is not work. Is it
possible to
> register the PSTN gateway as a user? How to do that?

That is strange. If you register the PSTN gateway as a user you can only
send requests corresponding to the PUID of the gateway to the gateway.

> I try to register a user with Xlite and PSTN gateway and compare them.
> Register user with X-lite is successful, but the PSTN gateway is not
successful.

The reason is that the username in the Authorization header field has to be
a NAI formatted identity and actually the gateway only uses the
username-part. The "@open-ims.test" is missing.
