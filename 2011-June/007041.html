> which one is Strange? I register the PSTN gateway to IMS, like IMS user
> register to IMS. Please explain, do I have wrong understanding in this
test ?
> Because I think, the PSTN gateway is like SIP phone. They convert the PSTN
> signalling into the SIP messages , after that register to the IMS.

No. A PSTN gateway is a trusted network node. It therefore does not need to
register, and it has to accept (forward) all requests corresponding to a
specific address range.
A registered user will only receive requests for one of its assigned
addresses (implicit registration set).
