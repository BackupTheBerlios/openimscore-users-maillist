I think you mean the Gentoo Linux VM image of Vladimir Romanov
[vromanov at gmail.com]

I found a description on that a few months ago:

1) Optimized for Core2. But I can recompile (make -e world). Portage tree in
this image deleted to preserve space.
2) root user password is password :)
3) to update OpenIMS core you can use /opt/OpenIMS/compile.sh. This script
will download new version from SVN and compile
4) To shrink image you can use ~/prepare.sh inside VM and later pack.cmd.
5) OpenIMSCore started startup. You can use Alt-F6,Alt-F7... To switch
between logs
6) This installation use testik.com as realm. This is normal domain, but
some host in this domain points to internal IPs (192.168.101.*)
7) named only used as DNS cache

To connect client to this server
1) Add some address from 192.168.101.* network to client PC
2) Setup IC_Lite and use attached config file.
Please edit this file (profile.cfg)
#replace USERNAME & PASSWORD with username & password #by default
password==username #copy this file to "C:\Program
Files\OpenIC_Lite\" directory
