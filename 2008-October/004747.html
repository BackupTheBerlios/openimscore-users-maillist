Hello,

The differences between an IMS SIP client and a non IMS SIP are (among 
others) :
- 3GPP Release 5 Requirements (RFC 4083)
- P-headers (RFC 3455/3325)
- AKA-MD5 (RFC 3310)
- Security agreements (RFC 3329)
- Service-route mechanism (RFC 3608)
- Reg-event package (RFC 3680)
- IPv6 and IPSec

This list is non exhaustive, but I hope it will help clarify the point.

You can also read this article which is a good introduction: 
http://www.tmcnet.com/sip/0307/feature_articles_sip_development_tool.htm

Regards, Laurent Etiemble.
