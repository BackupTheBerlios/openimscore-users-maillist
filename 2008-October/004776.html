Hi,

I am writing some plugins for Nagios, which we want to use to monitor 
OPENIMSCORE. So I tried to send an INVITE request to myself, but it
does not work. 

Normally, when I am receiving an INVITE request, I got 4 Record-Route 
Headers:

Record-Route: <sip:mt at pcscf.open-ims.test:4060;lr>
Record-Route: <sip:mt at scscf.open-ims.test:6060;lr>
Record-Route: <sip:mo at scscf.open-ims.test:6060;lr>
Record-Route: <sip:mo at pcscf.open-ims.test:4060;lr>

With this Record-Route Header everything is fine.

But when I try to call myself, the INVITE request has only 3 Record 
Route-Route Header:

Record-Route: <sip:mt at scscf.open-ims.test:6060;lr>
Record-Route: <sip:mo at scscf.open-ims.test:6060;lr>
Record-Route: <sip:mo at pcscf.open-ims.test:4060;lr>

When i am sending back an ACK following these routes, ACK is 
dropped. On the P-CSCF I am getting the message
"Orig_Subsequent: Request not following indicated dialog routes
 Orig_Subsequent: ACK not following dialog routes discarded silently!!!
"

I am using the default OpenIMSCore Configuration.

Is it possible to send an INVITE to myself with the default configuration?
