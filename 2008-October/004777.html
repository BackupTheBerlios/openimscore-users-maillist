Hi Johannes,

> But when I try to call myself, the INVITE request has only 3 Record
> Route-Route Header

The problem is that the terminating INVITE request is not recognized as an
initial terminating request, because a dialog has been created already by
the originating leg. Therefore it is handled as subsequent terminating
request (and no Route header is therefore inserted).

> Is it possible to send an INVITE to myself with the default configuration?
No.
