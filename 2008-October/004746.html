As for the Linux version, maybe one day if someone sponspors it... :-)

Anyway, here is the change log with all the technical details:
- Add ability to dial to an arbitrary contact
- Add iLBC codec (RFC-3951/RFC-3952)
- Add support for OMA Presence Rules (OMA-TS-Presence_SIMPLE_XDM-V1_1)
- Add support for OMA Presence Rules default handling policies 
(OMA-TS-Presence_SIMPLE_XDM-V1_1)
- Add support for PRACK in Audio/Video sessions (RFC-3262)
- Add support for network-initiated re-authentication (TS-24.229)
- Add support for handle 401/407 on re-INVITE
- Fix default value for Personal Privacy header (RFC-3323)
- Fix out-of-band DTMF timestamping in RTP (RFC-4733)
- Fix in-band DTMF clock-drifting in audio stream
- Fix 423 handling on REGISTER dialogs (RFC-3261)
- Fix redundant unsubscription on "reg" event package (TS-24.229)
- Fix failure on contact manipulation if XDM server does not advertise 
resource-list (RFC-4825)
- Fix failure on contact and presence manipulation if XDM server does 
not server correct documents (RFC-5025)
- Fix crash on non-exisiting end-point
- Fix crash on startup with an invalid XDMS endpoint
- Fix label clearing when returning from a locked session

Regards, Laurent Etiemble.
