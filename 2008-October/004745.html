> Mercuro Announcement wrote:
> > We are proud to announce that a new release of the Mercuro IMS Client is
> > available. To download the software installer, please go to
> > http://www.mercuro.net/ and follow the download link.
> >
> > Mercuro IMS Client is an 3GPP/IMS softphone and features:
> > - Unified Contact List Management (XCAP/XDMS)
> > - Incoming/outgoing Voice calls
> > - Incoming/outgoing Video calls (QCIF, CIF)
> > - Instant Messaging (SIMPLE, MSRP)
> > - File Transfert (MSRP)
> > - OMA Presence management
> > - Multi-codec support (G711, iLBC, SPEEX, AMR, H.263, H.264, ...)
> > - RCS compliance
> >
> > You must be aware that even if the Mercuro IMS Client is quite
> > functional, it is still in a beta stage. Anyway, we would like to hear
