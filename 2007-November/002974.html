Hi Frederic,

The dialog state (Call-id, from & to tags) seem to be correct. However, the
NOTIFY-request does not include the P-Asserted-Identity header which *may*
cause the P-CSCF to reject the request since the identity has not been
asserted.

Hope this helps!
