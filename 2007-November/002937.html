Hi,
 
Looking at the log you'll see a few lines which goes like
"java.io.FileNotFoundException: logs/hss.activities.log (Permission denied)"
 
This means that you do not have permissions to open and/or modify the file
logs/hss.activities.log. Check if the 'logs' directory exists and if proper
permissions has been set.
 
Shouldn't be very critical for the FHoSS anyways, but makes it harder for
you to debug when something goes wrong.
 
Best regards,
