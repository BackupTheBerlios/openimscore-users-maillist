Hello Veronica,

for killser - just copy the ser_ims/cfg/killser to a directory in your
system's path. Probably it would be a good idea to call it from the cfg
directory (we'll change the files in SVN)...

For ser - you need to compile ser_ims. Please revisit the installation
guide and make sure that the compilation step was successful. If not,
please see the requirements again and if still in trouble let us know
what the compilation error was.

Then we would recommend that you start them in different terminals. Our
target is that you use them to experiment with IMS, so it is quite
useful to see the output and status. If not, change the parameters given
to the ser from the provided example startup bash scripts, such that
they would start in the background. with stopser or killser you could
then do for example stopser icscf (this will stop all your I-CSCFs -
more specifically, the ones started with a configuration file that
contains "icscf" in the name).

Cheers,
-Dragos
