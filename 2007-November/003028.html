For one, it looks like you miss-configured the clients. The REGISTER
RequestURI should be "sip:open-ims.test" and not an IP Address. You are
registering to a domain, not to a node.

then if you configure the S-CSCF as it is by default to select AKA
authentication, x-lite won't work as it only knows to do MD5.

-Dragos
