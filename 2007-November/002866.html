Hi,

 

I believe that the FOKUS XDMS is closed source and that you can buy a
license, most likely a license for the code as well after signing a
non-disclosure agreement.

 

An other option is to use the OpenXCAP which is open source and seem to have
similar functionality or build your own from some Java Servlet
implementation. The FOKUS XDMS uses the FOKUS SIP Servlet Execution
Environment (SIPSEE) which is there own HTTP Servlet and SIP Servlet
implementation. They make use of Apache Xindice
<http://xml.apache.org/xindice/>  as stated in there desription.

 

The address to OpenXCAP is http://www.openxcap.org/

 

Best regards,

Mikael
