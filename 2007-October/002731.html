Hey Jin,

Just install :
- Hss on one machine,
- SER on three machines (for P-CSCF, I-CSCF and S-CSCF)
- Bind in the machine you want, (doesn't matter)
- and Mysql on the HSS and the I-CSCF.

Afterthat, you have to configure IP addresses
For that, there is a FOKUS's tool named configurator.sh. You could easily
changes IP adresses with that.

To configura HSS, go to the folder deploy in /opt/OpenIMSCore/FHoSS/deploy
and put IP Addresses in files where you have to do

Don't forget additionnal librairies to be added to the four machines.

And, enjoy :)
