> No, you are making a lot of sense. The think is though, that the cfg
> files that we provide are more of a guide on how you could make a CSCF,
> than a definitive one.
>
> So the I-CSCF does implement some basic NDS and THIG. Yet, by default,
> these are not enabled. Look a little in the icscf documentation and the
> extra cfg file provided and you should be able to figure out how you
> would build the required domain-border functionality. Also you could add
> proprietary functionality from this configuration script.
>
> Please understand that we tried first to give a simplistic example to
> users, that would work for most of them and give them an easy time
> starting it for the first time.
>
> If you do change the default I-CSCF for this purpose and you think that
> others could benefit, please contribute back.
>
> Cheers,
> -Dragos
