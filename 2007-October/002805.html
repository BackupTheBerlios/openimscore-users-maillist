You have to create trigger points for specific applications running on
your App servers.

 

Then you have to link these Service Profile Triggers to an IFC and set
it to the concerned App Servers.

 

You can use the FHoSS web console to create the IFC's, trigger points
and service profiles.

 

I hope this helps...
