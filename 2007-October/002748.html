Did you check if http://127.0.0.1:8080 <http://127.0.0.1:8080/>  is
responding? Sometimes, RH installer sets up wrong /etc/hosts file where
localhost gets mapped to the actual IP address of the system instead of
the loopback interface. Also check on which IP your S- and P- CSCFs are
listening for their Diameter interface (in scscf.xml and icscf.xml).

 

Hope this helps.

 

-David
