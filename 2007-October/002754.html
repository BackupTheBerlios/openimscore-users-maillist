Hi!

As far as I know, there is no Sh interface support in Sailfin. 
But I think that you don't need it at this step.
You need to configure service profile in FHoSS, which will forward
messages to your AS instance and to subscribe one of your
OpenIMSCore users for this profile.
You can use SimpleProxy example application from Sailfin web-site.
