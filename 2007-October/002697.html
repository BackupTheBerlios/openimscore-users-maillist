Dragos Vingarzan <Dragos.Vingarzan at fokus.fraunhofer.de> a écrit : Hello Hammou,

Seems like the OpenIC receives some REGISTER messages. The OpenIC is an
IMS client, not a registrar, so it does not respond to these requests.

Also, the 405 response is not handled because normally IMS networks
would allow REGISTER requests.

-Dragos
