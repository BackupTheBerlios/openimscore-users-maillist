Hello Saurabh,

In your trace, the P-Preferred-Identity is wrong. This is a
configuration issue. That value should be one of the values that you
find in the 200 OK for REGISTER in the P-Associated-URI header or else
it is consider an impersonation attack.

Cheers,
-Dragos
