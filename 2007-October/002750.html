Hi,

You could run the databases on any of your computers or a totally different
i.e. pc5 as long as your configuration is correct and you have access
through firewalls and MySQL is configured to allow connection from a
different host.

The easiest (no extra configuration, and fastest no extra communication
outside the pc) is to run
 - the icscf database on the same as the icscf (pc3 in your case)
 - the hss database on the same as the hss (pc1 in your case)

The hss_db.sql contains a database called hss_db and is the base structure
of the database for the hss.

The userdata.sql contains a basic configuration of the hss with users alice
and bob and also the open-ims.test network settings entered. If you want the
base configuration loaded you should insert this at the same pc as the
hss_db.sql.
