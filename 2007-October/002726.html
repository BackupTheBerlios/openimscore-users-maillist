Allesandro,
if you are doing 802.1x authentication, you are using  Wx interface 
between the AAA and the HSS to be able to attach supplicant to the 3GPP 
packet network. As far as I know, there is not Wx implemented in the 
FoHSS. Supplicant should be able to perform EAP-SIM or EAP-AKA based 
authentication and create IPSec SAs towards PDG. The  UE is not 
affected. Andrej
