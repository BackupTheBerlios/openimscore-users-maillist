Hi David,

Unfortunately we can not claim IPv6 yet due to lack of resources. We
estimated that it won't take much - 1, maximum 2 MM.

A lot of people asked about it. Yet nobody required it that bad to
actually spend the financial or man-power resources on it.

so:
- yes it is planned, but no clear date. financing would solve the issue.
- no. The underlying SER already supports IPv6.
- The Diameter stacks need some relatively small extensions for opening
also IPv6 sockets instead of IPv4 only.
- the IMS extensions to SER have never been checked for IPv6
compatibility, so most probably fixes will need to be performed on code
that, for example, looks for dots when parsing IP addresses.
- I did not try to use IPv6 even just for SIP because I am most certain
that it won't work 100%.

Cheers,
-Dragos
