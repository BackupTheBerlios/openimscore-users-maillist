> Can we use openimscore  as IP-SM-GW?
No. This is a different network node which is attached to IMS like an AS,
but it need different interfaces to the messaging platform. It is -  as 
the
name implies -  a gateway.

> One more thing .. how can we enable sms module in openimscore?

IP-SW-GW. Most probably it will not work
