The versions of the notifications should
 apply on a dialog basis.

Like I said before, a version of 0 after a version of 2 would just be
ignored by a good client, so the server should not do that, but always
send bigger versions than the previous. This is what I understand from
the RFC.

If a UE is power off it de-registers. Then the subscription is deleted,
as I said in one of my first replies.

-Dragos
