You can use Wireshark for that. Set it to capture on the “any” device
instead of eth0 or whatever you’re normally using.

 

Check the port numbers to know which node sends to which. Also keep in mind
that running all the CSCFs on the same machine means that most of them
cannot use the default ports. So filtering for “sip and diameter” will
filter out some of the traffic. As far as I know, you need to find a packet,
right-click it and select “Decode” and then select the correct protocol in
the list


 

Good luck!

Andreas
