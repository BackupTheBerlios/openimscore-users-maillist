^Hi,
 
> Is it possible to see/capture the complete sip messages being exchanged
> between i,p & s cscf , just like you can do so with wireshark on a network
> interface ?

Yes of course. I use tcpdump with the following command which captures all
packets except SSH and MySQL also in case of packet-fragmentation:
tcpdump -i any -s 3000 -w /var/tmp/tracefile.pcap not port 22 and not port
3306

You can view the packets in Wireshark.
